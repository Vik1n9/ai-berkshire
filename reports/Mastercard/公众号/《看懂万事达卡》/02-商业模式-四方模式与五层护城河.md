# 四方模式與五層護城河

## 四方模式：為什麼這門生意幾乎不可能被複制

全球支付市場年規模超過 100 萬億美元。萬事達卡作為其中的基礎設施層，不擁有任何資金、不承擔任何壞賬，卻拿走了整條鏈路中最穩定的一部分利潤。理解這背後的邏輯，需要從四方模式說起。

四方模式的參與者：

| 角色 | 誰 | 做什麼 | 承擔什麼風險 |
|------|---|--------|------------|
| 持卡人 | 你 | 刷卡消費 | 還款責任 |
| 髮卡行 | 你的銀行（如工行、招行、花旗）| 髮卡、授信、收利息 | **信用風險** |
| 收單行 | 商戶的銀行 | 為商戶提供刷卡服務 | 商戶違約風險 |
| 卡網路 | **萬事達卡**（或 Visa）| 授權、清算、結算 | **幾乎為零** |

關鍵在最後一列：**萬事達卡把"過路費"留給了自己，把信用風險和商戶服務分給了銀行。**

這是網路效應的自然結果——銀行願意承擔風險，因為接入萬事達卡意味著發出的卡能被全球 1 億個商戶接受。不接入，卡就是廢塑膠。

### 輕資產到什麼程度？

FY2025 資料：

| 指標 | 萬事達卡 | 含義 |
|------|---------|------|
| 淨利率 | **45.7%** | 每賺 100 美元收入，淨賺近 46 美元 |
| 運營利潤率 | **59.2%** | 運營效率極高 |
| 自由現金流 / 淨利潤 | **109%** | 淨利潤幾乎全部轉化為真金白銀 |
| ROIC | **43-50%** | 每 1 美元投入資本，賺回 43-50 美分 |
| 信用風險敞口 | **0** | 不放貸、不持有消費者債務 |

資料來源：[Mastercard FY2025 年報](https://s25.q4cdn.com/479285134/files/doc_financials/2025/q4/4Q25-Mastercard-Earnings-Release.pdf)、[ROIC 資料](https://www.gurufocus.com/term/roic/MA)

**反方觀點**：有人說"不承擔風險"也意味著"不掌握客戶關係"——持卡人和商戶的直接關係是跟銀行的，不是跟萬事達卡的。這話沒錯。但萬事達卡已經透過增值服務（欺詐檢測、資料分析、身份驗證）在向"直接價值交付"轉型——FY2025 增值服務收入佔比已突破 **40%**，增速 **23%**。這一點後面會展開。

## 五層護城河：深且多層

萬事達卡的護城河不是一條寬水溝——它是五層防禦工事疊加在一起。

### 第一層：網路效應（最強）

- 全球超過 **30 億張**萬事達卡在流通
- 超過 **1 億個商戶**接受萬事達卡
- 覆蓋 **200 多個國家和地區**

網路效應是雙向的：持卡人越多，商戶越願意接入；商戶越多，銀行越願意髮卡；銀行越發卡，持卡人越多。

要建一個新的全球支付網路，需要**同時**說服所有銀行、所有商戶、所有消費者——這是經典的"先有雞還是先有蛋"問題。過去 50 年，全球只成功建成了兩個跨國卡網路：Visa 和 Mastercard。沒有第三個。

**反方觀點**：中國銀聯按髮卡量是全球最大的卡網路。但銀聯 **90% 以上的交易量來自中國境內**，在海外商戶接受度遠低於 Visa 和萬事達卡。規模大不等於網路效應強——關鍵是**跨境互聯互通能力**。

### 第二層：轉換成本（很高）

一家銀行要從萬事達卡切換到另一個網路，意味著：
- 重新簽約、重新發卡（數千萬張卡的物理替換）
- 重新對接所有收單行和商戶的系統
- 消費者的自動扣款、訂閱服務、繫結支付全部失效

這個成本是**天文數字**。所以現實中幾乎從未發生過銀行大規模"叛逃"卡網路的案例。

### 第三層：監管壁壘

支付網路需要在全球 200 多個國家獲得**監管許可、反洗錢合規認證、PCI-DSS 安全認證**。這些許可證不是花錢就能買到的——它們是幾十年積累的結果，涉及與各國央行、金融監管機構的長期關係。

一個新進入者即使有技術能力，**拿齊全球許可證也需要十年以上**。

### 第四層：資料資產

萬事達卡每年處理 **1,755 億筆** 交易（FY2025），金額達 **10.6 萬億美元**。這些資料為其提供了：
- **欺詐檢測**：實時分析交易模式，準確率持續提升
- **消費者洞察**：SpendingPulse 等產品為政府和企業提供宏觀消費資料
- **精準營銷**：幫助商戶提升轉化率

資料的價值隨規模**指數級增長**——處理的交易越多，欺詐模型越準確，增值服務越有價值。這是後進入者無法複製的。

### 第五層：品牌信任

在支付領域，"信任"就是一切。消費者和商戶選擇萬事達卡，因為它代表"交易一定會完成、資金一定會到賬"。這種信任建立在**數十年無重大事故的運營記錄**之上。

## 飛輪效應：越轉越快

五層護城河構成了一個自我強化的飛輪：

```
更多持卡人 → 商戶更願意接受 → 更多商戶接入
    ↑                                    ↓
    ← 更好的使用者體驗 ← 更多增值服務 ←
```

而增值服務正在為飛輪**增加新的維度**：

| 增值服務型別 | 作用 | FY2025 表現 |
|------------|------|------------|
| 網路安全與欺詐檢測 | 降低商戶/銀行損失 | 增長最快板塊之一 |
| 資料分析（SpendingPulse） | 為客戶提供消費洞察 | 邊際成本極低 |
| 身份驗證 | 數字商務基礎設施 | 代理商務新場景 |
| 精準營銷 | 幫商戶提升轉化率 | 高復購率 |

FY2025 增值服務收入佔比已突破 **40%**，增速 **23%**，Q4 單季增速 **26%**。這意味著萬事達卡不再只是"收過路費"——它還在向透過收費站的人"賣導航、賣保險、賣資料"。

資料來源：[Mastercard Q1 2026 業績](https://www.tikr.com/blog/mastercard-q1-2026-earnings-revenue-hits-8-4b-eps-up-23)

## 一個關鍵問題：PayPal、Stripe 是競爭者嗎？

很多人把 PayPal、Stripe、Square 視為萬事達卡的競爭者。實際情況更微妙。

| 公司 | 表面角色 | 實際關係 |
|------|---------|---------|
| PayPal | "繞過卡網路的數字錢包" | 大部分交易底層仍是 Visa/Mastercard 結算 |
| Stripe | "線上支付基礎設施" | 是萬事達卡的**大客戶**，非替代者 |
| Square/Block | "線下 POS + Cash App" | 規模遠小於卡網路，且依賴卡網路結算 |

PayPal 確實在嘗試"閉環支付"（用銀行賬戶直接支付，繞過卡網路），也推出了 PYUSD 穩定幣。但截至目前，PayPal 的大部分交易仍然**最終透過 Visa/Mastercard 網路完成結算**。

**這就是網路效應的力量**：即使你想繞過萬事達卡，你的使用者和商戶大機率還是在用萬事達卡的網路。

**反方觀點**：長期來看，如果 PayPal、Apple Pay、Google Pay 等數字錢包持續推動賬戶對賬戶（A2A）支付，確實有可能蠶食部分卡網路交易量。這個風險存在，但速度遠比預期慢——因為消費者習慣和商戶系統的切換成本極高。

## 與 Visa 的雙寡頭格局

萬事達卡和 Visa 合計控制了全球**超過 80%** 的跨國卡網路支付量：

| 網路 | 全球卡網路市值份額 | 關係 |
|------|------------------|------|
| Visa | **43.9%** | 老大 |
| Mastercard | **36.4%** | 老二 |
| 中國銀聯 | ~15% | 主要在中國境內 |
| 美國運通 | ~4% | 高階小眾 |
| 其他 | <1% | 忽略不計 |

資料來源：[Capital One Shopping Research](https://capitaloneshopping.com/research/credit-card-market-share-statistics/)

雙寡頭格局的關鍵特徵：**這不是零和博弈**。Visa 和萬事達卡的真正對手是**現金**，而非彼此。全球無現金化滲透率每提升 1 個百分點，兩者都受益。它們更像是"同一條高速公路上的兩個收費站"——車流量增加對兩者都是好事。

**反方觀點**：雙寡頭格局也意味著監管風險——當兩家公司控制了太大的市場份額，政府可能會介入。歐盟已經實施了交換費上限，美國也有反壟斷審查的壓力。這個風險在第 04 篇詳細討論。

## 本篇小結

| 維度 | 判斷 |
|------|------|
| 商業模式 | 四方模式——把風險分給銀行、把"過路費"留給自己，輕資產到極致 |
| 護城河深度 | 五層疊加：網路效應 + 轉換成本 + 監管壁壘 + 資料資產 + 品牌信任 |
| 飛輪狀態 | 正在加速——增值服務增加了新維度 |
| 競爭者威脅 | 金融科技公司更多是客戶而非敵人——但長期 A2A 支付值得關注 |
| 雙寡頭格局 | 穩固，真正的敵人是現金，而非彼此 |
