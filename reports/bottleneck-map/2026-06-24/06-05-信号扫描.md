## 一、AIXTRON（AIXA.DE）— 新發現：InP/EML MOCVD裝置，Layer 3 AI供應鏈

### 為什麼值得關注（本輪觸發）

AIXTRON是生產EML鐳射器和PIC（光子積體電路）所必需的MOCVD（金屬有機化學氣相沉積）裝置的主要供應商。EML鐳射器已被Nvidia以$40億鎖定（Lumentum + Coherent），非Nvidia買家交期延伸至2027年以後。InP MOCVD裝置在這條供應鏈中位於**Layer 3**（生產EML鐳射器所需外延片的製造裝置）。

**Q1 2026關鍵資料（2026年5月披露）**：
- Q1 2026收入：**€59.4M**，同比下降47%（季節性 + 電力電子需求疲軟）
- Q1 2026**訂單量：€171M，同比增長30%**
- **>65%訂單來自光電子（Optoelectronics）**：G10-AsP平台，面向AI資料中心EML和PIC基鐳射器
- FY2026指引：**上調至約€560M**（原€520M），EBIT率目標17-20%
- 分析師評語：photonics bet正在兌現，收入滯後但訂單領先

### 供應鏈瓶頸定位

```
Layer 0：AI模型推理/訓練
Layer 1：GPU/AI加速器
Layer 2：光模組（400G/800G+），EML鐳射器
Layer 3（AIXTRON所在）：
  ├─ InP/GaAs外延片（MOCVD生產）
  └─ MOCVD裝置：Aixtron G10-AsP平台
Layer 4：InP/GaAs襯底，高純銦、鎵源
```

MOCVD裝置的瓶頸判定：
| 標準 | 評估 | 評級 |
|------|------|------|
| 供給集中度 | 全球僅Aixtron、Veeco（及有限的中國廠） | 🔴 ≤3家 |
| 擴產週期 | 裝置交期10-12個月，擴產需時 | 🔴 >1年 |
| 替代難度 | 中國廠商技術尚未達到InP光子學水平 | 🔴 難以替代 |
| 產能利用率 | Q1訂單爆發說明產能接近滿負荷 | 🟡 70-90% |
| 需求增速 | AI資料中心拉動，EML緊缺 | 🔴 >50%/年（光子學訂單） |
| 驗證週期 | 新MOCVD裝置需要1年以上工藝驗證 | 🔴 >1年 |

**瓶頸評級：A級（嚴重受限）**，4個🔴

## 三、彈藥/硝化棉危機 — Layer 3 國防供應鏈新訊號

### 瓶頸事實

歐洲彈藥生產鏈中的硝化棉（Nitrocellulose）出現結構性短缺，是2026年最被低估的國防供應鏈瓶頸之一。

| 資料點 | 數值 | 來源 |
|--------|------|------|
| 歐洲+烏克蘭年需求 | **~20,000噸** | EPC |
| 歐洲年產能 | **4,500-10,000噸** | EPC |
| 年度缺口 | **≥10,000噸（最大14,000噸）** | EPC |
| 中國控制棉絨份額 | **>70%（歐洲進口）** | 多源 |
| 中國對硝化棉+銻的限制 | **2025年起出口管制** | 多源 |
| BAE Systems替代（木漿） | 最早2026年底量產 | 多源 |

### 供應鏈Layer定位

```
Layer 0：155mm炮彈（烏克蘭戰場消耗）
Layer 1：彈藥生產（Rheinmetall, BAE, 北歐防務）
Layer 2：發射藥（Propellant）— 已有缺口
Layer 3（瓶頸）：
  ├─ 硝化棉（Nitrocellulose）← 中國出口管制
  └─ 棉絨（Cotton Linters）← 中國供應70%以上
```

**瓶頸評級：A級**（3個🔴：供給集中度、替代難度、需求增速）

### 已知可投資標的

| 公司 | 程式碼 | 市值 | PS | PE | 與硝化棉的關係 | 評判 |
|------|------|------|-----|-----|--------------|------|
| Rheinmetall | RHM.DE | €55.4B | 5.5x | 35x（forward） | 收購Hagedorn-NC，在瑞士/西班牙/南非生產NC，垂直整合 | 大市值，已充分定價 |
| Chemring | CHG.L | ~£1.6B | ~2.5x | ~20x | 彈藥/對抗系統，記錄訂單£1.364B，但**是NC的下游使用者，面臨原料約束** | 現貨受益者但也是受害者 |

**Rheinmetall估值檢查**：
- 市值€55.4B，PE forward 35x，PS 5.5x → 🟡 黃燈（PE>30x）
- Hagedorn-NC垂直整合是戰略差異化，但€55B大市值已將國防上升週期充分定價
- 10年25x PE退出：需要淨利潤從當前~€1.06B增長至約€2.2B（35%總增長×10年），增速要求低 → 但初始PE 35x對應10年後25x PE退出= 0.71x總回報（貶值）
- 實際計算：若淨利潤從€1.06B以15% CAGR增至€4.29B，25x PE = €107B，vs 市值€55.4B → 年化6.7%（仍低於10%）

**結論：彈藥/硝化棉是真實A級瓶頸，但現有大市值列表公司（RHM、BAE等）估值已反映國防上升週期。硝化棉純正受益者（Eurenco等）未上市。訊號歸入"需跟蹤，暫無合格標的"類。**
