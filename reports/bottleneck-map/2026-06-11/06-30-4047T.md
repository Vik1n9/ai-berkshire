#### 估值檢查（必填）

**PS計算**：
- 追蹤PS = ¥204.18B / ¥65.4B = **3.1x** 🟢
- 前向PS（FY2027） = ¥204.18B / ¥95B = **2.1x** 🟢

**PE計算**：
- 追蹤PE = ¥204.18B / ¥3.79B = **53.9x** 🟡（黃燈——需解釋）
- 前向PE（FY2027E） = ¥204.18B / ¥7.3B = **28.0x** 🟢（首次綠燈）

**紅燈條件核查**：
1. ❌ 市值>TAM 20%：WF6全球市場TAM約¥50-100B（估算），市值¥204.18B——約TAM的2-4倍，超過20%閾值？→ **邊界情況。** WF6是特氣中的細分品類，若以更廣泛的半導體特氣市場計（約¥1-2T），則無問題。WF6純粹TAM較窄但增長迅速（GAA時代需求翻倍），此條留黃燈標註不觸發封頂。
2. ❌ PS>30x且增速<100%：PS 3.1x，**不觸發**。
3. ❌ 市值>5年樂觀預測10倍：5年後樂觀收入（CAGR 20%，到¥163B）×10=¥1,630B >> 市值¥204B，**不觸發**。

**估值黃燈說明**（追蹤PE 54x）：
- 追蹤PE 54x觸發黃燈（>80x為紅，但54x需要解釋）
- 解釋：FY2026是產能爬坡年（淨利率5.8%偏低），FY2027E淨利率≈7.7%，邊際改善顯著；FY2027前向PE=28x是更具參考意義的估值指標
- PEG檢驗：前向PE 28x / 淨利增速+93% ≈ PEG **0.30**——非常低，增速超支援估值

**10年25x PE退出安全邊際**：
- 當前市值：$1.36B（¥204.18B / 150 JPY/USD）
- 10年10%年化目標：$1.36B × 1.1^10 = **$3.53B**
- 所需退出淨利潤：$3.53B / 25 = **$141M USD**（¥21.2B）
- 起點（FY2027E淨利潤）：¥7.3B（$48.7M）
- 所需CAGR：(21.2/7.3)^(1/10) - 1 = **~11.3%**
- 可行性判斷：FY2027已實現+93%淨利增速，後續在GAA需求驅動下維持11%長期CAGR是保守假設
- **結論：具備安全邊際。**

**估值最終判斷：🟢 綠燈。從上輪⚠️黃燈正式升級。**

#### 看多邏輯

1. **GAA結構性需求提升每晶片WF6消耗量**：Samsung/TSMC的3nm、2nm GAA電晶體相比FinFET需要更多層W金屬柵極CVD工藝，每片晶圓的WF6消耗量提升估計20-40%（行業估算）。即使晶片出貨量不增長，WF6需求也在結構性增長。
2. **前向PE 28x疊加53%利潤增速，PEG 0.30**：在這個增速下，28x PE實際上並不貴。對比國內化工同類公司，增速遠不及此但PE通常在20-30x。
3. **護城河深：三重壁壘**——特殊氟化工裝置（資本密集）、日本半導體供應鏈認證（多年建立）、WF6品牌信譽（半導體級純度要求極高，客戶不願意輕易更換）。

#### 看空邏輯

1. **股價已漲374%，好訊息大部分已被定價**：從¥796到¥3,560，前向PE 28x是否已包含FY2027全部指引？如果FY2027實現不達預期，回撥空間大。
2. **Air Products WF6業務擴張**：APD在WF6領域也有產能，且有意擴大半導體特氣業務。若APD顯著擴產，供給端壓力會減輕。
