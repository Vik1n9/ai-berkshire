| 紅燈條件 | 評估 | 結果 |
|---------|------|------|
| 市值 > TAM的20% | WF6全球市場約$300-500M（約3000-5000MT × $100K/MT）；Foosung市值$1.34B已超過WF6 TAM的20% → 若WF6是唯一驅動力則觸發紅燈 | ⚠️ 紅燈（TAM過小，但Foosung有冷媒/電池等其他業務） |

**黃燈審查（虧損 + PS < 15x）**：
- PS 3.8x，虧損 → 黃燈條件（虧損+PS>15x）未觸發
- 但最新季度虧損$9.96M，前一季度盈利$11.46M → 盈利波動性高，不穩定
- 若WF6價格傳導至收入，H2 2026有望轉盈利，但時間表不確定

**估值安全邊際（10年25x PE退出法）**：

| 專案 | 數值 |
|------|------|
| 當前市值 | $1.34B |
| 10年10%年化目標 | $1.34B × 2.593 = **$3.47B** |
| 所需淨利潤（25x PE） | $3.47B / 25 = **$139M** |
| 當前淨利潤 | 近零/虧損 |
| 關鍵問題 | Foosung年收入$355M，要達到$139M淨利潤需要約**39%淨利率**——這是特氣行業罕見的高利率 |
| 現實路徑 | WF6價格+70-90%，但原料（鎢粉）成本也在上漲；能否實現高利率存在顯著不確定性 |
| **結論** | **安全邊際不足**（需要WF6貢獻大幅提升 + 高淨利率，當前缺乏足夠資料支撐） |

### 看多邏輯

1. **WF6供應衝擊是客觀事實**：7/1日本雙廠永久停產，全球25%產能消失，認證週期18-24個月，短缺至少持續至2027年底。Foosung是有能力快速填補缺口的少數幾家供應商之一，已鎖定CSSC貨源認證。

2. **70-90%漲價已落地**：合同價格已確定，不是預期而是現實。2026年下半年營收和毛利潤應有實質改善。PS 3.8x在這個背景下相對合理。

3. **韓國政治地緣優勢**：中國鎢出口管制針對日本，韓國受影響較少，為Foosung提供相對優勢。

### 看空邏輯

1. **股價已反映大量預期**：YTD+196%。投資者早已"發現"WF6故事。在當前價格買入，需要相信未來12-18個月的基本面改善比市場更樂觀。

2. **WF6佔收入比例未知**：Foosung有冷媒、電池材料等多元業務。若WF6僅佔總收入20-30%（約$70-100M），70-90%漲價對EPS的提升可能沒有市場預期的那麼大。

3. **成本端也在漲**：鎢粉是WF6原料，同樣受中國出口管制影響。若原料成本漲幅超過售價，利潤率提升有限。最新季度已出現虧損印證這一風險。

### 結論與建議

**訊號強度：★★★**（有真實瓶頸 + 財務資料尚可 + 但股價已大幅定價 + WF6佔比不明 + 安全邊際不足）

**建議**：加入觀察名單，等待以下驗證再決定是否升級：
1. **Q2 2026財報（8月）**：確認WF6漲價是否已體現在收入和毛利中；WF6佔收入比例是否足夠
2. **8月大規模進口節點**：CSSC貨源認證是否如期完成
3. **7/1之後的價格動態**：日本停產落地後，Foosung的定價權是否進一步提升
