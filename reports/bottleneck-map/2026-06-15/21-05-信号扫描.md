## 訊號 A：中國放寬InP出口管制（2026-06-15，今日新訊號）

**來源**：Digitimes（China eases InP substrate exports lifting compound semiconductor supply 2026-06-15）、Futunn（DIGITIMES China approved batch InP substrate exports 2026-06-15）、Digitimes（China squeezes InP supplies jolting US-Taiwan optics chain 2026-06-11）、BigGo Finance（Indium Phosphide prices surge 250% China export controls 2026）

### 事件梗概

| 維度 | 內容 |
|------|------|
| **核心事件** | 中國於2026-06-15批准新批次磷化銦（InP）襯底出口許可；5月底首批2026年度貨物已完成發貨 |
| **背景** | 2025年2月起中國對InP實施出口管制；6英寸InP晶圓價格從$1,400飆升至~$5,000（+250%） |
| **促成原因** | 美中5月貿易談判期間，Anderson訪華時將InP出口許可證延遲列為重點議題，貿易談判後中國做出部分讓步 |
| **規模** | 目前僅為"批次"而非"全面解除"——許可制度仍在，但管控開始鬆動 |
| **比較基準** | 6/11，Digitimes仍報道"中國收緊InP供應，衝擊美台光學鏈"；6/15轉向"中國放寬" |

**與現有瓶頸邏輯的交叉驗證**：

- 本系統對InP/EML S級瓶頸的核心假設之一是"中國InP出口管制 → 西方供給無法快速填補"
- 今日新訊號顯示：**中國出口管制已開始軟化**，這是InP/EML S級瓶頸的重要"潛在降級訊號"
- 但：（1）許可證制度仍在，非永久性解除；（2）美中貿易關係仍有不確定性；（3）Ras Laffan損毀導致的製造業材料整體供需偏緊仍在

### 對各標的的影響

**AXTI（AXT Inc，納斯達克 ★★）**：
- 邏輯變化：AXTI子公司Tongmei是最大中國InP生產商，此前積壓$100M+因出口管制無法轉化為收入
- 今日訊號意味著Q2/Q3 2026收入可能出現明顯提速（積壓開始可出貨）
- 估值挑戰不變：市值~$5.1B，PS ~40x，紅燈仍在；但**基本面觸底反轉訊號增強**
- 結論：★★維持，但關注Q2財報（7月下旬）——若收入大幅超預期+PS降至<30x → 升級候選

**IQE plc（LON: IQE ★★★★候選）**：
- IQE生產外延晶圓（需要InP襯底投入）；中國InP襯底放寬 → 材料成本可能改善
- 但：中國InP襯底同時可供中國內部EML鐳射器製造商使用 → 可能擴大中國廠商競爭
- 淨效果不明確；IQE ★★★★候選評級不變，等8月中報確認>20%增速+EBITDA轉正
- 注意：FY2025 EBITDA僅£3.2M（同比-60%），FY2026需大幅改善才能晉升

**VECO（Veeco，納斯達克 ★★★）**：
- 中國InP放寬 → 短期內西方EML廠商擴產壓力略減 → 對VECO裝置需求節奏有輕微不確定性
- 但：Nvidia $4B鎖定COHR/LITE產能至2027+，這些公司仍需要MOCVD裝置擴產以完成承諾
- 淨效果：VECO需求不受實質影響，★★★維持；等股價回撥至$60-62區間

**InP/EML S級瓶頸整體評估**：

| 因素 | 變化方向 |
|------|---------|
| 中國InP出口許可（訊號A） | ⬆️ 供給鬆動（但許可制仍在，非永久解除） |
| Coherent 6英寸晶圓+產能翻倍（訊號C） | ⬆️ 西方供給中期擴張 |
| LightCounting：mid-2026緩解預期 | ⬆️ 市場共識看法 |
| Nvidia $4B鎖定COHR/LITE至2027+ | ⬇️ 供給端仍被大客戶壟斷 |
| 800G光模組產能缺口40-60%（至2027） | ⬇️ 需求側瓶頸仍在 |

**階段性判斷**：InP/EML S級瓶頸仍維持，但供給端多重改善訊號已出現。如果中國InP出口持續放寬+Coherent 6寸晶圓規模化，2026年底前瓶頸可能從S級降至A級。這是一個**下行風險增加的S級瓶頸**，而非堅不可摧的單點故障。
