## 硝化棉/棉短絨瓶頸詳解

### 供應鏈拆解

```
Layer 0：155mm炮彈 / 火箭彈藥
Layer 1：無煙火藥（Smokeless Powder）/ 推進劑
Layer 2（重點）：硝化棉（Nitrocellulose）— 本次新發現的瓶頸節點
Layer 3：棉短絨（Cotton Linters）— 原料；中國控制歐洲供給>70%
Layer 4：棉花加工副產品供應
```

### 瓶頸判定6條標準

| 標準 | 評估 | 評級 |
|------|------|------|
| 供給集中度 | 全球棉短絨高純來源：中國>70%（歐洲視角） | 🔴 |
| 擴產週期 | 新建硝化棉產能需3-5年（化工許可+建設） | 🔴 |
| 替代難度 | BAE開發無需硝化棉的推進劑，但技術成熟期2026年底，量產待定 | 🟡 |
| 產能利用率 | 歐洲存量產能已滿負荷執行，擴產專案大量開工 | 🔴 |
| 需求增速 | 俄烏戰爭+歐洲國防預算膨脹，彈藥需求持續加速 | 🔴 |
| 客戶驗證週期 | 軍用推進劑替換須經國防部長期驗證（1-3年） | 🔴 |

**瓶頸評級：S級**（5個紅燈）。但與WF6不同，短期內消除路徑更可見：Eurenco/Hanwha/波蘭三大擴產專案均已立項，2027-2028年有望大幅緩解。
