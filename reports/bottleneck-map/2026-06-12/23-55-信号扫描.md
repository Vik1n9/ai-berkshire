**$550M增發時間線**：
```

                當日AXTI股價+14%至約$89
```

**PS計算更新**（以增發後口徑）：
| 情景 | 市值 | 收入假設 | PS | 評級 |
|------|------|---------|-----|------|
| 當前（$89, ~63M股） | $5.6B | TTM~$90M（Q4 2025 ~$22.5M×4） | ~62x | 🔴 紅燈 |
| FY2026前向（$35M/季run rate） | $5.6B | $140M | ~40x | 🔴 紅燈 |
| FY2027前向（$65M/季run rate） | $5.6B | $260M | ~21x | 🟡 黃燈（但~3年後） |

**門檻價調整**（增發後，63M股口徑）：
- PS<15x（投資門檻）：$15 × $140M = $2.1B → 目標股價 $2.1B/63M = **≈$33**（vs舊口徑~$27）
- PS<10x（保守買點）：$10 × $140M = $1.4B → 目標股價 ≈ **$22**
- 即使FY2027收入兌現$260M：PS<15x = $3.9B / 63M = **≈$62**（接近當前價格，但需等到2027）

**結論**：
- 增發已兌現，稀釋已反映在市值中（+9.84M股）
- 120M授權暗示管理層預留了第二輪增發空間，若Tongmei Phase 2需要額外資本將再次稀釋
- Northland $125目標 = 前向FY2027 PS ≈ ($125 × 70M）/$260M = ~33x → 仍高於健康估值，但分析師認為可接受
- **評級維持★★（紅燈）**：當前PS~62x，門檻為PS<15x對應約$22-33股價（取決於FY2026/2027收入兌現時機）；寧可等看到$35M/季收入實際落地、且股價回撥至PS<20x時再考慮升級

**稀釋風險評分**：
- [x] $550M增發（~9.84M股）已兌現 → 已稀釋約15%
- [x] 120M授權透過 → 最多可再增發~57M股（巨大潛在稀釋空間）
- [x] 2個424B5 SEC檔案 → 兩次增發的註冊底稿（一次已完成，一次可能預備中）
- 結論：**稀釋風險持續活躍**，投資者需持續監控
