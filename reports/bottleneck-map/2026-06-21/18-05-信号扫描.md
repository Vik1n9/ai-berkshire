## 詳細分析：CLF/GOES訊號

### 為什麼值得觀察

**瓶頸鍊條**：資料中心建設激增 → 變壓器需求暴增 → 變壓器交期4年 → 核心原料GOES（粒取向電工鋼）供應不足 → CLF是**美國唯一國內GOES供應商**。
- 進口GOES面臨50%關稅（2025年起擴充套件至變壓器鐵芯及矽鋼疊片）
- 白宮將變壓器供應鏈列入DPA → GOES國產化需求政策加持
- Weirton變壓器工廠開業後，CLF將同時生產GOES和變壓器，垂直整合

**反面論據**：
1. GOES僅佔CLF產品銷量3%（Q1 2026），對CLF整體利潤影響有限
2. CLF是週期性鋼鐵公司，主營鐵礦石和平板鋼，估值受鋼鐵週期驅動
3. 若全球鋼鐵價格下行，CLF整體估值壓縮，即使GOES受益也被淹沒
4. 無純GOES上市標的——這是CLF的小部門，不是獨立公司

**估值快照**（CLF）：
| 指標 | 資料 |
|------|------|
| 市值 | $7.63B |
| Q1 2026營收 | $4.9B（年化約$19.6B） |
| PS | ~0.39x 🟢（整體超低估，但GOES佔比只有3%） |
| PE | 未知（CLF在2024-2025年有虧損歷史，2026年盈利狀況需核實） |
| 競爭地位 | 美國唯一國內GOES供應商 |

**初步結論**：CLF作為GOES/變壓器瓶頸的**間接受益者**，而非純正標的。整體鋼鐵業務主導定價，GOES受益尚無獨立顯現路徑。記錄為潛在訊號，需進一步拆分GOES業務利潤貢獻。暫不進入觀察名單正式排名。
