## 訊號詳析

### 訊號1：Centrus-Oklo LOI — 商業HALEU市場首個大規模訊號

**事件詳情（2026-06-18，SEC 8-K）**：
- Centrus與Oklo簽署非約束性意向書（LOI）
- Centrus承諾從俄亥俄州Piketon設施供應足夠驅動"最多5座Aurora Powerhouse"的HALEU
- 多年期供應，交付從2029年開始
- Oklo可能向Centrus預付款以支援燃料供應和園區建設
- 背景：Oklo計劃建設1.2 GW電力園區

**為什麼這是新維度**：

此前LEU的核心跟蹤邏輯是"DOE Option 1b二元事件（6/30截止）"——如果DOE不續約，Centrus的HALEU生產線將失去運營資金。Oklo LOI引入了一個新的邏輯分支：

```
原邏輯：
  DOE Option 1b決定 → 續約/不續約（二元事件）
  不續約 = 資金短缺 = 生產可能暫停

新邏輯補丁：
  商業HALEU客戶（Oklo）開始預付款 → 部分替代DOE資金 → 降低不續約的災難性風險
  但：2029交付意味著現在的現金流貢獻有限
  關鍵問題：Oklo預付款規模vs. Centrus運營資金需求？（LOI未披露數字）
```

**對LEU投資論點的影響**：
- ✅ 商業HALEU市場正在形成，不完全依賴DOE
- ✅ Oklo是Altman領導的AI+核能概念，有融資能力，LOI可信度高
- ⚠️ 但LOI是非約束性的，定義協議尚未簽署
- ⚠️ 2029交付意味著2026-2028期間的運營現金流依然依賴DOE續約
- ⚠️ DOE Option 1b決定（T-9天）仍是最關鍵的近期事件

**結論**：LEU維持★★★★★⚠️，但信心略增：即使DOE短暫不續約，商業軌道正在建立。建議在6/30 DOE決定後第一時間報告。

| 2026-06-15 | "中國放鬆InP襯底出口，提振化合物半導體供應" | Digitimes 2026-06-15 |

**解讀**：這是戰術性批次釋放，不是政策性鬆綁。

**機制分析**：中國每次批准一批配額而非開放自由出口，這種"批次審批"模式給市場製造不確定性，也給中國保留了隨時收緊的槓桿。Lumentum CEO明確表示"公司已售罄至2028年，儘管產能翻了四倍"——這與短暫放鬆一批貨不矛盾，因為放鬆的量相對需求仍然不足。

**對IQE.L的影響**：
- IQE.L是InP外延片（epi-wafer）製造商，位於Layer 2。其上游是InP襯底（substrate），中國管控的恰好是這一層
- 部分放鬆 → IQE.L的InP襯底原料供應改善（短期正面）
- 但IQE.L的競爭對手（主要是中國公司和Coherent自制）也受益於放鬆
- 淨影響：中性略正，不改變長期結構性增長邏輯

**IQE.L狀態**：★★★★ 維持，但標註"短期供應視窗已開"

**估值計算**：

- **PS**：€6,400M / €560M = **11.4x** 🟡 黃燈（>10x但<15x，收入增長）
- **EBIT預測**：€560M × 18.5% = €103.6M
- **淨利潤估算**：€103.6M × (1 - 25%稅) = **€77.7M**
- **PE**：€6,400M / €77.7M = **82x** 🟡 黃燈（略超80x閾值）
- **PEG**：PE 82 / 收入增速22%（FY2025→FY2026）= **3.7x** — 高
- **52周漲幅+319%** vs **分析師目標-25%**：市場情緒遠超分析師共識

**紅燈條件檢查**：

1. **市值 > TAM × 20%**：
   - MOCVD裝置全球市場：約$20-30億/年（2026E，包含LED/SiC/鐳射/RF等全部細分）
   - 專門針對InP/AI光通訊的MOCVD子市場：約$3-5億/年當前，增長中
   - 即使用寬口徑TAM（$30億）：20% = $6億。AIXA市值€6.4B ≈ $7B >> $6億
   - **紅燈觸發** ⚠️ → 訊號強度封頂★★

2. **PS > 30x 且增速 < 100%**：PS 11.4x — **未觸發** ✅

3. **市值 > 5年樂觀收入預測 × 10**：
   - 樂觀情景：FY2031收入€1.5B（5年CAGR 21%）
   - 10× = €15B vs. 市值€6.4B — **未觸發** ✅

4. **增發後60天內翻倍**：無確認的近期增發資訊 ✅

**估值10年25x PE安全邊際檢驗**：
- 當前市值：€6,400M
- 10%年化目標：€6,400M × 1.1^10 = **€16,600M**
- 所需淨利潤（25x PE）：€16,600M / 25 = **€664M**
- 當前淨利潤：€77.7M
- 所需增長倍數：8.5x in 10年 = **年化23.8%**
- 判斷：對於受制於SiC/EV下行週期的MOCVD公司，即使AI光通訊開啟新增量，要求10年持續23.8%利潤增長是極高標準。**無明確安全邊際。** ⚠️

**訊號強度：★★**（TAM紅燈觸發封頂。瓶頸邏輯真實，Layer 3地位真實，Lumentum大客戶確認——但當前估值已過度反映預期：股價+319%、分析師共識-25%、MOCVD TAM尺寸難以支撐€6.4B市值）
