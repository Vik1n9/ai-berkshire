## Coherent Texas動土：對IQE風險評估的修正

**前輪（第236輪）記錄**："Coherent在德克薩斯Sherman建$154M 6英寸InP晶圓廠，產能12個月翻倍"

**本輪修正（來源：Coherent官方、NVIDIA部落格、CHIPS法案公告 2026-06-16）**：

| 專案 | 前輪記錄 | 修正後 |
|------|---------|-------|
| NVIDIA投資額 | $1.54億 | **$20億**（3月宣佈） |
| CHIPS法案支援 | 未提及 | **$5000萬**（DoC LOI） |
| 總設施規模 | ~$1.54億 | **$6.5億** |
| 產能變化 | 翻倍（12個月） | **四倍**（12個月內） |
| 動土時間 | 未知 | **2026年6月16日** |
| 重要性 | Jensen Huang到場 | 確認 |

**對IQE ★★★★評級的影響分析**：

Coherent的Sherman擴產是否威脅IQE的Tower/MACOM客戶群？

**正面：IQE未被直接替代**
- Coherent的InP產能用於自身EML鐳射器生產，向NVIDIA供應垂直整合產品（光收發器）
- Tower Semiconductor（IQE的多年期合同客戶）是Coherent在光子IC代工領域的競爭對手，不會採購Coherent的epiwafer
- MACOM（IQE戰略投資方+長期協議）同樣與Coherent存在競爭關係
- 本質上：Coherent是競爭對手的垂直整合，而非IQE客戶的替代供應商

**正面：規模擴張驗證論文**
- Coherent願意在12個月內投入$20億+獲得四倍InP產能，證明AI資料中心InP需求缺口的緊迫性已達到Capex級別
- 2026-06-18同日：Tower+Marvell宣佈累計出貨500萬顆Coherent Photonic IC（Tower SiPho平台產能真實緊張，對IQE的需求繼續上升）

**負面：需監控的風險**
- 如果Coherent未來開放epiwafer外銷（非主要業務，但不能排除）
- 12個月內（約2027年6月）全球6英寸InP epiwafer供給將因Coherent而顯著增加，可能觸發IQE定價壓力
- $20億級投資的Coherent規模遠超$596M市值的IQE，競爭資源不對稱

**結論：IQE評級維持★★★★**。Coherent擴產是競爭對手垂直整合，非直接客戶替代，且確認了InP瓶頸的真實性。但需要監控Coherent是否出現外銷epiwafer的動向。將"Coherent垂直整合威脅"的風險等級從"已知但規模有限"上調為"已知且規模重大（四倍產能，$20億資本）"。

## Foosung（093370.KS）初步估值核查

**瓶頸定位**：韓國氟化學/特氣公司，是WF6（六氟化鎢）的韓國主要供應商之一。日本Kanto Denka+Central Glass 7/1永久停產（約佔全球25%產能），Foosung+SK Specialty填補缺口。

**當前財務資料**（來源：StockAnalysis.com；Investing.com；HTX/KuCoin 2026-06）：

| 指標 | 資料 |
|------|------|
| 市值 | ~$13.4億（約1.9萬億韓元） |
| 年收入（TTM） | ~$3.55億 |
| PS | $13.4億 / $3.55億 = **3.77x** 🟢 |
| PE | 資料不足，需核實 |
| 股價近期變化 | 受WF6漲價訊息推動，具體漲幅未核實 |
| WF6業務佔收入% | 不明（Foosung同時涉及汽車化學品、電池材料、製冷劑） |

**估值初步判斷**：
- PS 3.77x 屬於合理水平（綠燈），未觸發紅燈條件
- 但關鍵問題：**WF6業務佔Foosung總收入的比例未知**。如果WF6只佔20%收入，實際受益彈性大幅打折
- 另一個問題：WF6供應缺口是否是結構性瓶頸？中國CSSC產能已達2000噸/年，Foosung的8月認證完成後，市場可能有更多替代供給，漲價週期可能較短

**初步結論**：PS合理，但WF6業務純正度不足（非單一產品公司），且供給缺口可能在6-9個月內因中韓產能認證而部分緩解。建議加入觀察名單進行進一步核實，**不作為明確標的**。
