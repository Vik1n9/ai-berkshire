### 訊號1：ALM $700M可轉債超額認購今日交割 — Phase 2融資鎖定

**核心事實**（來源：Globe and Mail, TipRanks, Investing.com, BusinessWire）：

| 條款 | 數值 |
|------|------|
| 本金 | US$700,000,000 |
| 利率 | 2.25%，半年付，2027年1月1日起 |
| 到期 | 2031年7月1日 |
| 轉換價 | $27.40/股（36.495股/$1,000本金） |
| 認購狀態 | **超額認購（oversubscribed）** |
| 交割日期 | **2026年6月9日**（今日） |
| 超額配售權 | 額外$1億，13天內行使 |
| 上限認購 | Cap Call上限$41.36/股（收盤價~$20.68時的100%溢價） |
| 用途 | 再融資現有債務 + 支援鎢礦擴張（Phase 2） |

**ALM當前財務狀態**（來源：Yahoo Finance, Macrotrends）：

| 指標 | 數值 |
|------|------|
| 當前股價 | ~$16.80（2026-06-07，較CB定價日~$20.68下跌約19%） |
| 市值 | ~$45.7億 |
| 流通股 | ~2.837億股 |
| 潛在新增股（全轉換）| 700,000 × 36.495 = ~2,555萬股（若$100M超額行使則+~365萬股） |
| 潛在稀釋比例 | ~9%-10%（若全部轉換，但需股價超$27.40才觸發） |

**狀態變化分析**：

- 此前狀態：Phase 2融資待定 + AGM投票不確定性
- 當前狀態：**融資已鎖定，AGM投票變為程式性事項**
- $700M = 當前市值的15.3% → 較大比例，但CB結構下不立即稀釋
- 當前股價$16.80遠低於轉換價$27.40，CB目前完全不在錢裡（OTM 38.7%）
- Cap Call上限$41.36保護了$27.40-$41.36區間的潛在稀釋

**風險變化**：

| 風險 | 變化前 | 變化後 |
|------|--------|--------|
| 融資風險 | 高 | 基本消除（$700M已到賬） |
| 執行風險 | 存在 | 仍然存在（擴產需2027年完工） |
| 稀釋風險 | 一般 | 已知且可量化（最多~10%，需觸發$27.40） |
| 債務償還風險 | 低 | 上升（2031年需還$700M+，但有5年時間視窗） |

**結論**：Phase 2決策已從"是否做"變為"怎麼做"。融資層面風險消除，執行層面開始計時。AGM Phase 2投票結果（預計6/10 EST早盤前公告）仍需等待，但結果已失去決定性意義。

### 訊號2：TSMC CEO 6月4日年度大會 — AI晶片短缺"將持續數年"

**核心宣告**（來源：TechTimes 2026-06-05, TechZine, MoneyCheck）：

- **C.C. Wei在年度大會（2026-06-04）明確表示**：AI晶片短缺將持續"數年"，不會像記憶體晶片那樣出現價格急劇波動
- **定價**：TSMC已開始上調最先進製程（3nm及以下）定價，H2 2026 3nm製程漲價約**15%**，2027年再漲5-10%
- **營收展望**：2026年營收增速仍維持30%+預期（需求強勁）

**對已跟蹤標的的影響**（交叉驗證）：

| 標的 | 影響 |
|------|------|
| VECO（MOCVD裝置）| ✅ AI資本開支援續 → InP鐳射擴產需求持續 → H2 2026訂單釋放邏輯維持 |
| 3110.T（T-glass）| ✅ AI伺服器+PCB需求持續 → T-glass短缺持續驗證 |
| 4047.T（WF6）| ✅ 晶圓製造量持續上升 → WF6需求持續 |
| CHG.L（彈藥）| 間接利好（NATO防務開支與AI無關，獨立邏輯） |

**結論**：這是截至6月9日最權威的AI供應鏈tightness一手確認，有效交叉驗證所有AI基礎設施瓶頸論點，沒有改變評級，但增加了論點可信度。

### 訊號3：固體火箭發動機 — 兩項新政府行動

**新資料**（來源：The Defense Post 2026-02-24; StockTitan 2026-05-26; executivegov）：

1. **DARPA選擇RTX/Raytheon開發可組合固體火箭發動機**（2026-05-26）：
   - 目標：開發模組化SRM技術，解決供應鏈單點故障問題
   - 意義：政府層面承認SRM供應鏈的結構性缺陷，開始"繞過"AMPAC單點故障尋路
   - 對CHG.L：中性偏正面（長期替代方案開發，短期供應仍依賴現有產能）

2. **美國國防部新增$3270萬DPA Title III SRM投資**：
   - 目標擴大國內SRM工業基礎的關鍵產能瓶頸
   - 繼2月Anduril $4370萬之後的第二筆DPA大額投入

**對CHG.L的影響**：邏輯進一步鞏固。短期（1-3年）內，政府擴產投入反映SRM供給嚴重不足，不是供給過剩訊號。中期（3-5年），DARPA模組化SRM可能帶來技術路線變化（風險）。
