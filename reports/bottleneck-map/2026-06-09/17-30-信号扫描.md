### ⬆️ 狀態變化1：LEU（Centrus Energy）— 風險框架重大更新，從"二元風險"到"近期暫停 / 長期管道強健"

**觸發事件**：本輪掃描首次發現2026年1月 DOE $2.7B鈾濃縮擴產任務單中，Centrus獲$900M用於擴建Piketon, Ohio設施商業級HALEU產能，另有$2.3B商業LEU承購承諾。此前各輪掃描聚焦"Option 1b 6/30截止"二元風險，未捕獲這兩項重要資料點。

**新資料（來源：IndustrialInfo、Centrus PR、SEC 10-K 2026-02-10）**：

| 專案 | 內容 |
|------|------|
| DOE $900M HALEU擴產任務單 | 2026-01-05宣佈，用於在Piketon, Ohio建設商業級HALEU濃縮產能；與原HALEU Operations Contract獨立 |
| $2.3B商業LEU積壓 | 公用事業客戶的購買承諾（含國內+出口），待融資確認 |
| 全年收入指引（2026） | $450M–$500M（中值$475M） |
| 市值（2026-02-20） | 約$4.01B；當前股價~$338（近90天跌12%），估算現市值約$3.5B |
| PS（估算） | ~$3.5B / $475M = **約7.4x** ✅ 綠燈 |
| Q1 2026淨利潤 | $10M GAAP（$0.51 EPS），Adjusted $23.5M（$1.19 EPS） |

**風險框架更新前後對比**：

| 維度 | 更新前 | 更新後 |
|------|--------|--------|
| 近期（21天）核心風險 | Option 1b不行權 → 產線中斷 | 不變：Option 1b中斷風險仍存在 |
| 若中斷的後果 | 嚴重，收入幾乎全來自HALEU運營合同 | **緩和**：有$900M擴產合同、$2.3B商業積壓提供後續收入通道 |
| 長期收入可見度 | 僅靠DOE HALEU運營合同續期 | **大幅改善**：$900M任務單 + $2.3B商業積壓構建多年收入管道 |
| 商業模式轉型 | 單一DOE承包商 | 向商業LEU/HALEU供應商轉型 |

**但另一方面（反向論證）**：
1. $2.3B商業積壓**有條件**：須先完成新產能融資建設，非確定收入
2. $900M DOE任務單仍處於"subject to negotiations"階段（2026-01-05原文），正式合同尚未完全簽署
3. Option 1b若6/30前未行權，HALEU運營中斷仍可能導致短期股價下行
4. LEU股價近90天跌12%，市場部分反映了這種不確定性

**結論**：維持 ★★★，但下行風險比此前評估的更小。若6/30前無Option 1b公告而股價進一步下跌，可能出現更好買點。監控優先順序維持 🔴。
