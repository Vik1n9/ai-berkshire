**估值檢查（4.2.1節）更新**：

- PS 3.3x → 綠燈 ✅
- PE 50.1x < 80x → 未觸發黃燈。但3.3x同業溢價 = 已有顯著重估。
- PE vs同業5x判斷：50.1x / 15.1x = 3.3x（未達"5倍同業"黃燈觸發線75.5x） → 正式標準下無黃燈
- 市值>TAM 20%？：全球WF₆ TAM約$2-3B，20%=$400-600M。市值$1.03B > $600M → **⚠️ 邊界觸發**。但關東電化非純WF₆公司，電池材料為第二主業，且WF₆ TAM因價格+232.7%動態擴大 → 標註"邊界觸發，等WF₆業務佔比核實"

**10年25xPE退出安全邊際**：
- 當前市值：$1.03B
- 10年10%年化目標：$1.03B × 1.1^10 = **$2.67B**
- 25x PE所需淨利潤：$2.67B / 25 = **$106.8M**
- 當前淨利潤：$20.7M（需5.2倍增長）→ CAGR **18%** over 10年
- 關鍵矛盾：7/1停產意味著WF₆產量驟降 → 悲觀情景下淨利潤下降 → **18% CAGR無保障**
- 樂觀情景：WF₆價格+232.7%驅動存量庫存/短期過渡合同毛利暴漲，隨後從非中國來源（越南/葡萄牙）重建供應鏈，中期ASP鎖定高位 → 3-5年內淨利潤有望達$50-80M（前向PE 13-21x）

**看多邏輯（保持）**：
1. WF₆是半導體刻蝕工藝不可替代氣體，韓國/歐美無短期替代來源（中國替代需6-18個月認證）
2. 7/1停產→全球WF₆缺口≥2,200噸/年（25%全球產能），價格將在三星/SK Hynix/TSMC需求壓力下持續飆升
3. 關東電化若能從非中國來源恢復部分產能，將以3-4x ASP復產，ROE大幅改善

**看空邏輯（本輪新增）**：
1. **PE 50.1x已反映預期**：WF₆危機已公開數月，市場已給出3.3x同業溢價。如果危機實際衝擊營收（7/1後無產品可賣），估值回撥風險大
2. **停產=營收消失**：WF₆產量為零，即便單位價格暴漲，如無庫存可出售，季收入可能腰斬
3. **電池材料業務緩衝**：如果電池材料（第二主業）能維持利潤，安全邊際提高；但細分貢獻不詳

**評級調整**：★★★★★⚠️ → **★★★★**（PE已知，無正式紅黃燈，但停產-收入博弈需7/1後第一份季報驗證。升回★★★★★需看到WF₆停產後關東電化仍錄得淨利增長）

- 虧損 + PS > 15x → **黃燈觸發**：允許進入★★★，但須說明盈利路徑和時間表
  - 路徑：Sangdong滿產能(~6,000噸/年WO3) → 年化收入$180-230M（當前~$100M）→ EBITDA轉正 → 盈利預計2027H2-2028H1
- 市值>TAM 20%？：全球鎢市場約$2-5B/年，20%=$400M-1B。市值$5.06B > $1B → **觸發紅燈**
  - 但：Sangdong是全球最大原生鎢礦之一，中國出口禁止推動非中國來源TAM擴大。此紅燈有"TAM在動態重估中"的註腳

**10年25xPE退出安全邊際**：
- 當前市值：$5.06B
- 10年10%年化：$5.06B × 1.1^10 = **$13.12B**
- 25x PE所需淨利潤：$13.12B / 25 = **$525M**
- 今天EBITDA季率$6.1M → 年化$24.4M。須增長21.5x到$525M
- 即使Sangdong滿產能年收入達$250M，以鎢礦~30-40%淨利率 = $75-100M淨利潤
- $75-100M vs 所需$525M → **年化回報 < 10%，不具備安全邊際**
- 結論：**當前價格$5.06B不具備安全邊際**，屬於"tungsten稀缺敘事溢價"

- 實際被動需求約$4-5B × 0.04% = **$4.9-5B × ALM權重** → 數百萬美元至數千萬美元被動買單
- 這是短期技術性催化，不改變基本面安全邊際

**評級調整**：★★★★ → **★★★**（**⚠️市值>TAM紅燈觸發**，PE退出安全邊際不足。Russell 1000是技術催化，但基本面不支援$5.06B。僅保留作為"tungsten稀缺敘事跟蹤標的"，等Sangdong產能資料驗證時重新評估。）
