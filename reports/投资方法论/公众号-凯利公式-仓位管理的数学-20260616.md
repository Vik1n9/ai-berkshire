# 每筆都賺錢，長期卻破產——投資中最反直覺的數學陷阱

假設有這樣一個投資機會：每次下注，贏的機率是60%，贏了賺一倍，輸了賠光本次下注。也就是說，每投100塊，數學上平均能賺20塊——這是一個明顯對你有利的遊戲。

如果你每次都把全部資金押上去呢？贏三次，資金翻8倍——但只要輸一次，就清零了。而40%的失敗機率意味著，連贏的次數越多，那一次致命的失敗就越確定會到來。玩得足夠久，破產是必然的。

這不是極端假設。1998年，長期資本管理公司的合夥人包括兩位諾貝爾經濟學獎得主，他們的策略確實能賺錢——但25倍槓桿下，一次俄羅斯債務危機就讓46億美元灰飛煙滅。他們的錯誤不是方向看錯了，而是押得太重了。

問題不在於"該不該投"，而在於"該投多少"。六十多年前，一個資訊理論學者給出了精確答案。

---

## 凱利公式：到底該押多少

1956年，貝爾實驗室的物理學家約翰·凱利（John Kelly）解決了一個問題：在一個勝率和賠率已知的重複博弈中，每次應該押多少比例的資金，才能讓長期財富增長最快？

答案用一句大白話說：**用贏面減去輸面，再除以賠率，就是你該押的比例。**

寫成公式是：

> 最優倉位 = （賠率 × 勝率 - 敗率）÷ 賠率

用字母表示就是 f = (b×p - q) / b，其中 f 是最優下注比例，b 是賠率（贏了賺多少倍），p 是勝率，q 是敗率（1 - p）。

回到開頭的例子：贏賺一倍（b=1），勝率60%（p=0.6），敗率40%（q=0.4）。代入：f = (1×0.6 - 0.4) / 1 = 20%。凱利公式告訴你，每次押總資金的20%，是長期增長最快的策略。

再換一個：贏賺兩倍（b=2），勝率50%。f = (2×0.5 - 0.5) / 2 = 25%。賠率越好，可以押更多。

押少了，增長慢。押多了呢？不是增長更快——而是增長變慢，甚至倒退。這才是凱利公式最反直覺的地方。

## 超過上限，必然毀滅

![凱利曲線](../assets/kelly/fig1-kelly-curve.png)

下面這張表是全文最重要的資訊：

| 下注策略 | 長期增長率 | 波動率 | 意味著什麼 |
|---------|-----------|--------|-----------|
| 半凱利（0.5倍） | 全凱利的75% | 全凱利的50% | 用一半波動換四分之三的增長 |
| 全凱利（1倍） | 100%（理論最優） | 100% | 增長最快但顛簸極大 |
| 1.5倍凱利 | 全凱利的75% | 全凱利的150% | 增長率和半凱利一樣，波動卻是三倍 |
| **2倍凱利** | **零** | 全凱利的200% | **長期增長率為零，白忙一場** |
| **超過2倍** | **負數** | 更高 | **長期必然破產** |

請注意1.5倍那一行：增長率和半凱利完全一樣，但波動率是半凱利的三倍。過度下注不僅沒好處，還要承受大得多的痛苦。

2倍凱利更極端——每筆交易都有正期望，但長期增長率為零。超過2倍，即便每筆都賺錢，長期也必然虧光。

這就是開頭那個悖論的數學答案。

華爾街用真金白銀驗證過兩次：

**長期資本管理公司（LTCM），1998年。** 25比1的槓桿，四個月虧掉46億美元，差點引發全球金融系統崩盤。聯準會協調十四家銀行才收場。

**比爾·黃的Archegos基金，2021年。** 透過衍生品建立極高槓桿，峰值管理360億美元。兩天之內，歸零。給銀行造成超過100億美元損失。黃本人後被判十項罪名。

## 為什麼虧損50%比你想的嚴重得多

過度下注之所以致命，根源在於虧損的數學不對稱性。

![虧損不對稱性](../assets/kelly/fig2-loss-asymmetry.png)

| 虧損幅度 | 回本需要漲多少 |
|---------|-------------|
| -10% | +11% |
| -20% | +25% |
| -30% | +43% |
| -50% | **+100%** |
| -75% | **+300%** |
| -90% | **+900%** |

虧10%，漲11%就回來了，看起來差不多。但虧50%，需要翻倍才能回本。虧75%，需要漲三倍。虧90%——需要漲九倍。

原因很簡單：投資是乘法不是加法。虧50%是乘以0.5，漲50%是乘以1.5，0.5×1.5=0.75，不等於1。虧損縮小了本金，從更小的基數上漲回同樣的絕對值，需要更大的百分比。

**這就是凱利公式存在的數學原因：它天然避免了大幅回撤。** 凱利公式最大化的是對數增長率（幾何平均），而不是算術平均。對數增長率天然懲罰大幅波動——因為在乘法世界裡，一次大虧會抵消很多次小贏。

所以凱利公式不是一個"幫你賺更多"的公式，而是一個"幫你不虧太多"的公式。這兩件事在長期複利中是同一件事。

## 大師們的實戰：看懂了，才敢下重注

如果凱利公式只是數學遊戲，它不會引起投資者的興趣。但巴菲特、芒格、段永平的實戰操作，幾乎就是凱利公式的活教材——雖然他們可能從未用過這個公式。

先看一個投資版的凱利公式。當收益不是簡單的"贏/輸"而是連續分佈時（股票就是這樣），凱利公式變為：

> 最優倉位 = （預期收益率 - 無風險利率）÷ 波動率的平方

用字母表示就是 f = (預期收益 - 無風險利率) / 方差。大白話：**超額收益越高、波動越小，就應該倉位越重。**

這個公式還有一個資訊理論的解讀：你的最優增長率等於你相對市場的資訊優勢。如果你對一家公司的理解不比市場更深，最優倉位是零。

現在用這個框架審視大師們的操作：

![大師重倉案例](../assets/kelly/fig3-master-cases.png)

**巴菲特買美國運通，1964年。**

1963年運通因"色拉油醜聞"股價腰斬。巴菲特親自去餐廳和商店觀察，發現人們照常使用運通卡——生意沒有受損，只是市場恐慌。

用凱利框架分析：巴菲特判斷運通的內在價值遠高於市價（預期收益極高），而他透過實地調研確認了生意沒有受損（在他眼中波動很小）。公式的分子大、分母小，凱利值很高。他投入約1300萬美元，佔合夥基金的40%。結果：獲利數倍。

**段永平抄底網易，2002年。**

網易因財務問題股價跌到不足1美元，面臨退市。段永平研究後判斷網易的遊戲業務被嚴重低估。他湊了200萬美元（含100萬借款），在0.5-0.8美元之間買入。

用凱利框架分析：段永平判斷網易退市機率很低、遊戲業務價值遠高於市價（預期收益極高）。他自稱"看懂了"這門生意（在他眼中波動接近零）。凱利值支援重倉。結果：網易最高漲到100多美元，獲利約130倍，賺了大約2億美元。

**段永平買蘋果，2011年。**

以約13.75美元均價買入，持有14年幾乎沒動過。賺了約18倍。

這三個案例有一個共同點：**他們不是在賭，而是在資訊優勢極大的前提下按邏輯加倉。** 看懂了生意，波動在他們眼中就不是風險，而只是價格的搖擺。

芒格說得直接："過度分散的想法是瘋狂的。廣泛分散化必然包括投資平庸企業，只能保證平庸的結果。"

段永平更樸素："看懂生意比較難，看不懂的拿住比較難，大部分人拿不住是因為看不懂。"

用凱利公式翻譯這兩句話：**看懂了，波動就小，凱利值就高，就該集中。看不懂，波動就大，凱利值就低，就該分散或者不碰。**

## 半凱利：實戰中真正該用的策略

理論上全凱利增長最快。但幾乎所有嚴肅的實踐者都推薦只用一半。

**第一，你的引數估計一定有誤差。** 在賭場裡，勝率和賠率可以精確計算。在股市裡，預期回報和波動率都是猜的。如果你高估了自己的優勢，全凱利就變成了超凱利——而我們已經知道超凱利的下場。半凱利給誤差留了緩衝區。

**第二，增長率的損失遠小於波動率的降低。** 半凱利只犧牲25%的增長率，卻降低50%的波動。用75%的增長換50%的波動——這筆交易極其划算。

**第三，心理承受力有限。** 全凱利在極端情況下可能回撤50%以上。半凱利把回撤控制在人類心理可以承受的範圍內。

數學家羅坦多和索普的研究證實：**半凱利有約90%的機率避免毀滅性虧損，全凱利只有50%。**

說到索普——他可能是凱利公式最偉大的實踐者。這位MIT數學教授1962年出版了《擊敗莊家》，用凱利公式指導21點下注，被賭場列入黑名單。之後轉戰華爾街，創立基金連續19年年化回報15%-20%，230個月中只有3個月虧損。1973-74年大熊市標普跌42%，他逆勢盈利。

索普本人也推薦半凱利。理論上的最優和現實中的最優，中間隔著引數誤差、黑天鵝事件和人的心理極限。

## 不虧錢，才是最好的賺錢策略

回到開頭的問題：為什麼每筆都有正期望，長期卻可能破產？

因為投資不是單次博弈，而是無限次重複博弈。在重複博弈中，決定最終結果的不是算術平均（期望值），而是幾何平均（複合增長率）。凱利公式最大化的正是幾何平均。

具體到實戰，三條原則：

**第一，沒有資訊優勢，就不要下注。** 凱利公式清楚地說：當你對一家公司的理解不超過市場共識時，最優倉位是零。這和段永平"不懂不碰"完全一致。

**第二，即使有優勢，也只用半凱利。** 你對自己優勢的估計幾乎必然偏高。用一半倉位，給錯誤留夠空間。

**第三，永遠不要超過凱利上限。** 這是不可逾越的紅線。超過2倍凱利，不管分析多正確，長期結果都是歸零。LTCM和Archegos不是敗給了判斷力，是敗給了倉位。

巴菲特說投資的第一條規則是"不要虧錢"，第二條是"不要忘記第一條"。這兩條規則聽起來像廢話，但凱利公式用數學證明了它是最深刻的投資智慧。

> 投資比拼的不是誰賺得最猛，而是誰能留在牌桌上。活下去，比什麼都重要。
