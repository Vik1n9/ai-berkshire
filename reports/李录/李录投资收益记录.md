# 李錄投資收益記錄

> 喜馬拉雅資本（Himalaya Capital）1997年創立，總部西雅圖。管理資產約$150億（2025年）。不收管理費，前6%回報免費，超額部分收25%業績報酬。

**資料來源**：SEC 13F季度報告、港交所權益披露、證券時報深度覆盤、聰明投資者、財聯社等公開資訊。

---

## 一、核心戰役：比亞迪（最成功的投資）

這是李錄投資生涯中最濃墨重彩的一筆，也是價值投資教科書級的案例。

### 完整時間線

| 時間 | 事件 | 詳情 |
|------|------|------|
| **2002年7月** | 比亞迪港股上市，李錄首次買入 | 比亞迪H股上市價約10.95港元，後回落。李錄在約8港元左右建倉，成為基石投資者，持股約1.67% |
| **2003年** | 向芒格介紹比亞迪 | 李錄與芒格在感恩節初次長談後，開始向芒格推薦比亞迪。芒格起初不感興趣 |
| **2005年** | 年報顯示持倉899.75萬H股 | 佔比亞迪H股約1.67% |
| **2008年** | 大幅增持至5220.42萬股 | 佔比亞迪總股本約2.55%。同年金融危機，比亞迪股價暴跌 |
| **2008年9月** | 促成巴菲特投資比亞迪 | 李錄持續說服芒格，芒格最終"拍桌子"讓巴菲特買入。伯克希爾旗下中美能源以8港元/股買入2.25億股（10%股權），投資18億港元 |
| **2010-2020年** | 長期持有不動 | 比亞迪股價波動劇烈（2009年衝到88港元后回落到14港元），李錄始終持有 |
| **2021年7月** | 首次減持 | 分兩次減持1077.15萬股，套現超24.39億港元。減持後持倉約7365.06萬股 |
| **2021-2022年** | 持續小幅減持 | 比亞迪股價從200+港元衝至300+港元期間逐步減倉 |

### 收益估算

| 指標 | 資料 |
|------|------|
| 買入成本 | 約8-15港元/股（多年分批買入），總成本估計不超12億港元 |
| 持倉高峰市值 | 比亞迪最高約340港元/股時，持倉市值超250億港元 |
| 已實現收益 | 減持套現超180億港元 |
| 投資回報 | **超15倍**（成本計），若算最初8港元買入則**超40倍** |
| 持有時間 | **約20年**（2002-2022年） |

### 投資邏輯

李錄多次公開表達對比亞迪的看法：

"比亞迪是一家真正的工程師文化公司。王傳福本人就是工程師出身，十萬理工男的執行力是它的核心競爭力。"

"我當初買入比亞迪，看中的是它在電池技術上的積累和垂直整合能力。後來它切入汽車、新能源，都是這種能力的自然延伸。"

芒格評價："比亞迪是我見過的最了不起的公司之一。"

巴菲特後來也說過："在做出投資決定時，芒格只有兩次和我拍了桌子，讓我買買買。" 其中一次就是比亞迪。

## 五、投資風格總結

### 2018年前後的風格轉變

| 維度 | 2018年前 | 2018年後 |
|------|---------|---------|
| 市場重心 | **中概股為主**（百度、新浪、比亞迪） | **轉向美股藍籌**（谷歌、美國銀行、伯克希爾） |
| 持倉集中度 | 高 | **極高**（前3佔83%+） |
| 換手頻率 | 相對較高（平均15個月） | 極低（幾乎不動） |
| 投資型別 | 科技+中國消費 | 美國科技平台+金融 |

### 核心特徵

1. **極度集中**：通常只持有5-8只股票，前3大佔比超80%
2. **長期持有**：一旦看準，可以持有20年不賣（比亞迪）
3. **高勝率**：13只公開持股中11只賺錢，84.6%勝率
4. **果斷止損**：看錯了就走，不戀戰（拼多多第一次、阿里巴巴）
5. **危機中出手**：金融危機加倉比亞迪、疫情中買美國銀行、矽谷銀行危機買華美銀行
6. **不收管理費**：只在為投資者賺錢時才賺錢

### 李錄自述投資原則

"長期研究，合適時機，合適價格。"

"我們在有魚的地方釣魚。不需要理解所有公司，只需要真正理解少數幾家。"

"最好的投資是你永遠不需要賣出的投資。"

## 六、投資回報推算

### 整體基金回報（推算）

喜馬拉雅資本不公開整體回報率，但可從以下資料推算：

- **1997年成立**，至今28年
- **僅比亞迪一筆**即貢獻180億+港元收益
- 美股持倉從2019年$4.2億增長到2025年$35.7億（6年增長約8.5倍）
- 總管理資產約$150億

有分析師估算李錄的長期年化收益率在**20-30%區間**，與巴菲特早期合夥基金表現相當。
