## AI研究偏見自覺

**資訊豐富度評級：A級（資訊充裕）**

邁威爾科技1995年成立、2000年上市，44位分析師覆蓋。近兩年因定製ASIC/AI晶片概念大熱，媒體和賣方關注度飆升。2026年6月22日剛納入標普500指數。

**AI研究陷阱**：市場敘事高度一致——"AI基礎設施的隱形贏家"、"定製ASIC雙寡頭之一"。44位分析師中38位給"買入"，但當前股價$281已超過共識目標價$242。這種"分析師一致看好但股價已經跑在前面"的情況，意味著好訊息已經充分定價。

**應對策略**：核心問題——**$281是否已經把"定製ASIC黃金時代"的故事完全定價？邁威爾的競爭地位有多穩固？** 重點做反面檢驗：Trainium3競標失敗意味著什麼？客戶集中度風險有多大？

## 第二步：生意本質分析 — 段永平"對的生意"

### 一句話定義

**邁威爾是一家為超大規模雲客戶（AWS/Google/微軟）提供定製ASIC晶片和資料中心互聯晶片的"幕後軍火商"——不賣最終產品，而是為AI基礎設施提供關鍵零部件。**

### 商業模式畫布

```
核心引擎                     增長飛輪                    護城河來源
┌──────────────┐      ┌──────────────┐      ┌──────────────┐
│ 定製ASIC設計   │      │ 雲客戶AI投資  │      │ 設計鎖定      │
│ AWS Trainium  │<─────│ 加速→更多需求 │      │ 2-3年開發週期  │
│ Google Axion  │      │ 更多定製晶片   │      │ 切換成本極高   │
│ MSFT Maia     │      └──────────────┘      └──────────────┘
└──────────────┘
┌──────────────┐      ┌──────────────┐
│ 光互聯DSP     │      │ 儲存控制器    │
│ PAM4/PAM6    │      │ PCIe/NVMe   │
│ 70%市場份額   │      │ 傳統強項      │
└──────────────┘      └──────────────┘
```

### 生意本質的三個層面

**第一層：定製ASIC（核心增長引擎）**
- 為AWS設計Trainium AI訓練/推理晶片
- 為Google設計Axion ARM伺服器CPU
- 為微軟設計Maia AI加速器
- 商業模式：NRE（非經常性工程費）+ 量產版稅，單個專案生命週期3-5年

**第二層：光互聯DSP（現金牛+壟斷地位）**
- PAM4/PAM6光電訊號處理晶片，連線資料中心內GPU/CPU
- 市場份額約70%，業界首款3nm 1.6T晶片（Ara DSP）
- 每個AI叢集都需要大量光互聯→AI基礎設施的"賣鏟人"

**第三層：儲存/網路/交換機晶片（傳統業務）**
- 儲存控制器、乙太網交換機晶片、DPU
- 成熟但增長較慢，貢獻穩定現金流

### 毛利率與同行對比

| 公司 | Non-GAAP毛利率 | 說明 |
|------|---------------|------|
| 英偉達 | ~75% | GPU壟斷溢價 |
| 博通 | ~77% | 軟體+定製ASIC雙引擎 |
| AMD | ~54% | CPU+GPU競爭激烈 |
| **邁威爾** | **59.5%** | **定製ASIC毛利率低於標準產品** |
| 高通 | ~57% | 手機SoC競爭壓力 |

邁威爾59.5%的毛利率在半導體行業中處於中等偏上。低於博通和英偉達的原因是定製ASIC業務本質上是"設計服務"——客戶擁有最終晶片的大部分IP，邁威爾賺的是設計和製造管理費，天然毛利率低於自有IP產品。

### 段永平式追問

**這門生意好在哪？一句話：** 邁威爾卡在了AI基礎設施的兩個關鍵瓶頸——定製ASIC（替代通用GPU降低成本）和光互聯（連線GPU叢集的"神經纖維"）。只要超大規模客戶繼續擴建AI基礎設施，邁威爾就有確定性需求。

**但反面追問**：這門生意有一個根本性弱點——**客戶就是競爭對手的甲方**。AWS、Google、微軟隨時可以在下一代晶片中選擇其他設計合作伙伴（已經發生：Trainium3據報輸給了Alchip）。定製ASIC是"專案制"而非"產品制"，每一代都要重新競標。

## 第三步：護城河評估 — 巴菲特"經濟護城河"

### 五類護城河逐一驗證

| 護城河型別 | 具體表現 | 強度 | 趨勢 |
|-----------|---------|------|------|
| **品牌/定價權** | 在定製ASIC領域品牌效應不強——客戶看重的是技術能力而非品牌。定價權有限，大客戶議價能力極強 | ★★☆☆☆ | 穩定 |
| **轉換成本** | 單個ASIC專案2-3年開發週期+深度定製→當代產品鎖定極強。但代際切換時客戶可以選擇其他供應商 | ★★★★☆ | 代內強/代際弱 |
| **網路效應** | 不存在傳統網路效應。但積累的設計IP和經驗形成"學習曲線效應"——做過的專案越多，下一個專案效率越高 | ★★☆☆☆ | 緩慢變寬 |
| **規模效應** | 研發支出$17億/年，規模在定製ASIC領域僅次於博通。但遠小於博通（$90億+研發），規模劣勢明顯 | ★★★☆☆ | 穩定 |
| **技術/專利壁壘** | 光DSP 70%市場份額+3nm製程領先+SerDes/互聯技術積累深厚。5nm/3nm定製ASIC設計能力全球僅2-3傢俱備 | ★★★★☆ | 變寬中 |

### 護城河的核心矛盾

邁威爾的護城河存在一個結構性矛盾：

- **光互聯DSP**：護城河很深——70%市場份額，技術領先1-2代，客戶切換成本高。這是一個"產品型"業務，類似英偉達GPU的地位。
- **定製ASIC**：護城河是"專案型"的——每一代晶片都需要重新贏得客戶信任。Trainium3競標輸給Alchip證明，即使做了前幾代，下一代也不保證留住。

**Trainium3事件的重要性**：AWS的下一代AI訓練晶片Trainium3據報選擇了台灣的Alchip而非邁威爾。這意味著：(1)定製ASIC不是"一次贏得終身客戶"；(2)更小的競爭對手可以在特定專案上勝出；(3)邁威爾的ASIC護城河比市場認知的更脆弱。

### 巴菲特式追問

**10年後這條護城河還在嗎？** 光互聯DSP的護城河大機率在——資料中心對頻寬的需求只增不減，而PAM技術積累有很高的進入門檻。定製ASIC的護城河不確定——10年間會有多次代際切換，每次切換都是風險點。如果邁威爾在2-3次競標中落敗，這條"護城河"可能快速消失。

**什麼能摧毀它？** (1)博通在光DSP領域追上來（博通有資源和意願）；(2)定製ASIC大客戶在連續2-3個專案中選擇其他供應商；(3)超大規模客戶自建晶片設計團隊（蘋果模式），不再需要外部ASIC合作伙伴。

### 偏誤自查

- **敘事偏差**："AI賣鏟人"敘事極具吸引力，容易忽視定製ASIC是"專案制"而非"產品制"的本質差異
- **錨定效應**：股價從2024年低點$61漲至$281（+360%），容易產生"還能漲"的慣性。但$61對應的是AI還未爆發時的定價
- **倖存者偏差**：我們看到邁威爾的成功，但忽視了Acacia Communications（被思科收購）、Mellanox（被英偉達收購）等同類公司獨立存活的困難

### 芒格式追問

**我最可能在哪裡犯錯？** 最可能高估定製ASIC業務的"經常性"收入特徵。市場把邁威爾當作"AI基礎設施的經常性收入公司"來定價（Forward PE 62x），但實際上定製ASIC是專案型業務——每一代都要重新競標。Trainium3的失利不是意外，而是這種業務模式的固有特徵。

**聰明人為什麼不買？** (1)Forward PE 62x遠高於英偉達（21x）和博通（25x），但英偉達和博通的競爭地位都比邁威爾更穩固；(2)Non-GAAP毛利率從65%降至59.5%，定製ASIC佔比越高→利潤率越低；(3)股價$281已超過44位分析師的共識目標價$242。

## 第六步：行業與文明趨勢 — 李錄"文明演進框架"

| 領域 | 趨勢 | 邁威爾的位置 |
|------|------|------------|
