## 一家"無聊"到年賺 41 億美元的公司

每家公司都有一條命脈絕對不能斷——**發工資**。遲發一天，員工電話打爆 HR；報錯一筆稅，聯邦稅務局的罰單隨時來。這件事容錯率為零，卻幾乎沒有人知道背後是誰在做。

那家公司叫 **ADP**（Automatic Data Processing），1949 年成立於新澤西，主營一件事：**替企業發工資、報稅、管人力資源**。它不上頭條，不造概念，但全美超過 100 萬家企業每個月的薪酬週期都壓在它的系統上。看完下面這組數字，你就明白這台機器有多值錢：

| 指標 | 數值 | 量級判斷 |
|------|------|---------|
| FY2025 總收入 | **206 億美元** | 全球薪酬處理行業第一 |
| FY2025 淨利潤 | **41 億美元**（+9%） | 持續增長 |
| FY2025 EPS | **$9.98**（+10%） | 連續多年雙位數增長 |
| ROIC | **42%** | 全行業頂級 |
| 客戶留存率 | **92%** | 客戶幾乎不走 |
| 連續增加股息 | **51 年** | "股息貴族"標杆 |
| 服務企業數 | **超 100 萬家** | 覆蓋 140 多個國家 |
| 客戶資金餘額 | **約 483 億美元** | 類保險浮存金 |
| 當前股價（2026年5月） | **~$220** | 52 周高點 $330，低點 $188 |
| 當前市值 | **~880 億美元** | |
| PE（TTM） | **~22 倍** | 近十年偏低水平 |

（資料來源：ADP FY2025 年報、ADP Q3 FY2026 財報、Yahoo Finance）

把這些放在一起看——**年賺 41 億美元、ROIC 42%、連續 51 年提高股息、客戶留存率 92%，PE 只有約 22 倍，股價從高點回落超過 30%**。

好公司碰上了打折價格？還是一台正在減速的老機器？

這個系列只做一件事：**把 ADP 拆明白，讓你自己判斷值不值**。

## 三個關於 ADP 的常見誤解

### 誤解一："ADP 就是個發工資的軟體"

這是最表面的認知。ADP 確實幫企業發工資——但它做的遠不止這些。

ADP 的收入來自三層結構：

| 收入層 | FY2025 規模 | 本質 |
|--------|------------|------|
| 處理費（按人頭/按次收費） | 佔總收入主體 | 企業每月必須支付的"水電煤" |
| 浮存金利息 | **~12 億美元**（+16%） | 客戶資金在發放前暫存 ADP，ADP 用這筆錢賺利息 |
| 增值服務（PEO、合規、福利） | 持續增長 | 在薪酬基礎上疊加更多 HR 服務，提升客單價 |

第二層最有意思。ADP 手上常年趴著約 **483 億美元**的客戶資金——這些是企業交給 ADP 的薪酬和稅款，在合規期限到來之前暫存在 ADP 賬上。ADP 用這筆錢買債券賺利息，FY2025 光利息就賺了 **12 億美元**。

這個模式像誰？**巴菲特的伯克希爾·哈撒韋**。保險公司收保費、暫時不用賠付的錢拿去投資——ADP 收薪酬稅款、暫時不用發放的錢拿去賺利息。都是"用別人的錢賺錢"。

所以 ADP 不是"一個發工資的軟體"。更準確的定義是：**一台按人頭收費的合規處理機器 + 一家類保險浮存金公司**。

**反方觀點：** 浮存金利息高度依賴利率環境。如果聯準會大幅降息，這 12 億美元利息收入會縮水。ADP 採用階梯式投資策略（部分資金鎖定長期高利率債券）來對沖，但無法完全消除利率風險。FY2026 指引的利息收入為 **13.4-13.5 億美元**，說明管理層預計短期內仍有增長空間。

### 誤解二："薪酬處理門檻很低，AI 會把 ADP 幹掉"

大模型能算工資嗎？理論上能。但**算對工資和合規發工資是兩回事**。

薪酬處理的難點不在"算"，在"合規"：

- 美國 50 個州、數千個地方轄區，稅法各不相同且頻繁變更
- 140 多個國家的勞動法、社保、公積金規則完全不同
- 發錯一次工資面臨的不僅是罰款，更是員工信任的崩塌

ADP 幾十年積累的合規資料庫，不是一個大模型能替代的。而且 ADP 自己也在用 AI 強化產品——2025 年推出的 ADP Assist 能自動檢測薪酬異常、自動更新合規規則，早期使用者報告每個薪酬週期節省 **30 分鐘以上**。Q3 FY2026 財報顯示，AI 工具已將客戶來電量降低了 **8%**。

**短期看，AI 是 ADP 的加速器，不是替代品。**

**反方觀點：** 長期看（10 年以上），如果 AI 能力持續飛躍，出現"一鍵合規"的通用工具，ADP 的合規壁壘可能被侵蝕。科技巨頭也可能以 AI 為切入點進入薪酬市場。顛覆機率目前較低，但不可忽視。

### 誤解三："ADP 是成長股"

不是。ADP 是一台**穩定的複利機器**，不是高增長標的。

FY2025 收入增長 **7%**，FY2026 指引 **6-7%**。增速不性感，但勝在穩定。更重要的是，ADP 靠三條腿走路：

1. **自然增長**：經濟增長 → 企業招更多人 → ADP 按人頭多收費
2. **提價能力**：每年溫和提價 3-5%，客戶很少因為價格離開
3. **增值服務滲透**：在薪酬基礎上賣 HR 外包、福利管理、合規諮詢

這種增長結構的好處是：**不需要大筆資本投入，絕大部分利潤都能轉化為現金**。FY2025 全年自由現金流達 **47.8 億美元**，同比增長 **20%**。

ADP 不追求 30% 的收入增速——它追求的是**確定性**。7% 的收入增長 + 持續的利潤率擴張 + 連續 51 年提高的股息 = 一台長期複利機器。

**反方觀點：** 正因為增速只有 7%，ADP 的估值天花板有限。如果市場持續偏好高增長標的，ADP 可能長期被忽視。而且 7% 的增速如果受到經濟衰退衝擊（企業裁員 → 按人頭收費減少），可能短期滑落到 2-3%。

## 一張圖看懂 ADP 的賺錢邏輯

把 ADP 的商業模式濃縮成一個迴圈：

```
企業客戶 → 每月支付薪酬處理費（按人頭）
         → 同時把薪酬/稅款資金暫存 ADP
         
ADP → 處理薪酬和稅務合規（賺處理費）
    → 用暫存資金買債券（賺浮存金利息）
    → 在薪酬基礎上賣增值服務（提升客單價）
    
結果 → 輕資產、高 ROIC、現金流強勁
    → 客戶留存 92%、平均使用 11 年
    → 連續 51 年增加股息
```

這個迴圈最精妙的地方在於：**客戶越多 → 合規資料越全 → 系統越準確 → 吸引更多客戶**。這是一個正反饋飛輪。100 萬家企業客戶、140 個國家的合規資料，是任何競爭對手從零開始都需要數十億美元和十年以上時間才能複製的壁壘。

## 這個系列要回答什麼

ADP 不是一家有爭議的公司——幾乎所有人都同意它是好生意。真正的問題是：

1. **好到什麼程度？** 護城河是在加寬還是在收窄？
2. **競爭格局如何？** Paychex 合併 Paycor、Workday 虎視眈眈，ADP 的龍頭地位還穩嗎？
3. **增長從哪裡來？** 7% 的增速能維持多久？
4. **當前價格合不合理？** PE 22 倍、股價從高點回落 30%，這是機會還是陷阱？

| 篇目 | 核心問題 |
|------|---------|
| 01 開篇（本篇） | ADP 是什麼？三個常見誤解 |
| 02 商業模式 | 薪酬 SaaS + 浮存金 + PEO，三層收入結構深度拆解 |
| 03 競爭與增長 | Paychex、Workday、Paylocity，ADP 憑什麼贏？ |
| 04 風險與估值 | AI 替代、經濟週期、當前估值判斷 |

每篇不預設看多看空，先擺事實，再推邏輯。
