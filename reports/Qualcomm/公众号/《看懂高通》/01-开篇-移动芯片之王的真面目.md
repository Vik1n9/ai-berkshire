## 一組讓你重新認識高通的數字

全球每年賣出大約 12 億部智慧手機。絕大多數旗艦安卓機裡，有一塊驍龍處理器；而無論這塊處理器是不是高通的，手機廠商幾乎都要向高通另外付一筆錢——專利授權費。這就是高通的生意：既賣晶片，又收"過路費"。

這個認知不算錯，但還遠遠不夠。先看一組數字：

| 指標 | 資料 |
|------|------|
| 當前股價（2026年5月） | **~$209** |
| 市值 | **~2,170 億美元** |
| FY2025 總營收 | **443 億美元**（+13.7%） |
| FY2026 上半年營收 | **228 億美元** |
| QTL 專利授權稅前利潤率 | **超過 63%** |
| FY2025 自由現金流 | **128 億美元** |
| 5年平均 ROIC | **29.0%** |
| 汽車晶片年化收入 | **超過 50 億美元**（FY2026 Q2 已突破） |
| 汽車設計贏單管線 | **450 億美元** |
| PE (TTM) | **~21倍** |
| 股息率 | **~1.8%** |
| 年度回購+分紅 | **積極回饋**（200 億美元新回購計劃 + 季度股息 $0.92） |

（資料來源：Qualcomm FY2025 10-K、Q1/Q2 FY2026 財報、公開市場資料）

把這些數字放在一起，你看到的不是一家"手機晶片公司"——而是一家**同時經營"晶片設計"和"專利收費站"兩門生意的科技巨頭**，正在把觸角伸向汽車、PC、物聯網和端側人工智慧。

## 高通到底是什麼？

最常見的誤讀是把高通等同於"驍龍處理器"。驍龍確實是高通最知名的產品，但高通的商業模式比"賣晶片"複雜得多。

高通同時經營兩塊截然不同的業務：

### QCT：晶片技術（佔收入約 85%）

設計和銷售手機處理器、基帶晶片、射頻前端、汽車晶片、PC 處理器、物聯網晶片。FY2025 收入約 **377 億美元**。這是一門需要持續鉅額研發投入、面對激烈競爭的重資產生意。

### QTL：技術授權（佔收入約 15%，但貢獻鉅額利潤）

基於高通在 3G/4G/5G 領域積累的標準必要專利，向全球手機廠商收取專利許可費。FY2025 收入 **64 億美元**，稅前利潤 **40 億美元**，利潤率超過 **63%**。

這兩塊業務的本質完全不同：

| 維度 | QCT 晶片 | QTL 專利授權 |
|------|---------|------------|
| 性質 | 賣產品 | 收"過路費" |
| 利潤率 | 中等（受競爭擠壓） | 極高（>63%） |
| 護城河 | 中等（需持續研發領先） | 極深（標準必要專利無法繞開） |
| 增長空間 | 取決於新市場開拓 | 有限（繫結手機出貨量） |
| 類比 | 更像英偉達 | 更像高速公路收費站 |

**理解高通的關鍵在於：這兩塊生意形成了一個飛輪。** 晶片研發不斷產生新專利，充實專利池；專利授權的高利潤為高風險晶片研發提供穩定資金。砍掉任何一塊，另一塊都會顯著受損。

（資料來源：Qualcomm FY2025 10-K）

## 三個關於高通的常見誤判

### 誤判一："高通就是賣手機晶片的"

手機晶片確實是高通最大的收入來源——FY2026 Q2，QCT 手機業務季度收入約 **60 億美元**。但如果你只看到這一塊，你會錯過幾個重要的變化：

- **汽車晶片**：FY2026 Q2 季度收入 **13.3 億美元**（同比 +38%），年化已超 **50 億美元**，管理層目標是 FY2026 退出時年化 **60 億美元**、FY2029 達到 **80 億美元**
- **物聯網**：FY2026 Q2 季度收入 **17.3 億美元**（同比 +9%）
- **PC 處理器**：驍龍 X2 系列在 Windows 筆記本市場佔據約 **10%** 份額，超過 100 款設計方案進入量產或開發階段
- **專利授權**：每年穩定貢獻 **40 億美元+** 稅前利潤

非手機業務的收入佔比已從 3 年前的約 **30%** 提升到約 **40%**。高通正在從"手機晶片公司"變成"智慧計算平台公司"——這個轉型的成敗，決定了它未來 5 年的故事。

（資料來源：Qualcomm Q2 FY2026 財報、PCWorld 報道）

### 誤判二："蘋果自研基帶，高通完了"

這是投資者討論高通時出現頻率最高的看空論點。事實部分是對的，結論部分需要仔細算。

**事實**：蘋果 C1 自研基帶晶片已於 2025 年搭載在 iPhone 16e 上。2026 年秋季，C1 將進入更多 iPhone 17 機型。2026 年推出的 Ganymede 基帶將支援毫米波。到 2027 年，蘋果預計完全停用高通基帶。

**影響量級**：高通每年從蘋果獲得的基帶晶片採購收入約 **57-59 億美元**（注：不同來源口徑不同，底稿中用的 20 億美元為保守估計，最新資料顯示含 RF 元件在內的總金額更高）。這部分收入將在 2-3 年內歸零。

**但有一個關鍵的"但是"**：蘋果自研基帶不影響專利授權費。蘋果每賣出一部 iPhone，仍然需要向高通的 QTL 部門支付專利許可費（每台約 **8 美元**），因為 3G/4G/5G 標準必要專利無法繞開。

**反方觀點的反方**：汽車晶片的增長速度目前快於蘋果收入的流失速度。FY2026 汽車年化收入已超 50 億美元，而且設計贏單管線 450 億美元意味著這條增長曲線還遠未見頂。管理層對蘋果脫鉤有充分準備，這不是一個"突然被捅一刀"的風險，而是一個"已知的、正在對沖的"風險。

（資料來源：AppleInsider、Qualcomm Q2 FY2026 財報電話會議、Futurum Research）

### 誤判三："半導體公司都是週期股，不值得長期持有"

這個判斷對大多數晶片公司成立，但對高通需要打個折扣。原因在於那個利潤率超 63% 的專利授權業務。

看 FY2023 的資料：那一年全球手機市場去庫存，高通總收入暴跌 **19.2%**。但 QTL 專利授權收入僅小幅下滑——因為專利費是按手機出貨量收的，即使手機賣不動，只要還在生產，高通就能收錢。

**QTL 是高通的"反週期緩衝墊"。** 它讓高通在行業下行時不會像純晶片公司（比如聯發科）那樣劇烈波動。

但這個緩衝墊不是無限的。QTL 收入增長基本見頂（繫結全球手機出貨量約 12 億部/年），真正的增長引擎必須來自 QCT 的新市場。

（資料來源：Qualcomm FY2023-FY2025 年報）

## 看懂高通，需要回答四個問題

如果非要用一句話定義高通，最接近的版本可能是：

> **高通 = 一個利潤率 63% 的無線通訊收費站（QTL）+ 一台在手機、汽車、PC、物聯網四條賽道同時競賽的晶片設計機器（QCT）+ 一個正在失去最大客戶（蘋果）但試圖用新客戶填補缺口的轉型故事。**

要看懂這家公司，至少需要回答四個問題：

| 問題 | 這個系列哪一篇回答 |
|------|-------------------|
| "晶片+專利"雙輪驅動到底怎麼賺錢？驍龍生態有多強？ | 第 02 篇 |
| 端側 AI、汽車晶片、PC 處理器——新賽道能撐起增長嗎？ | 第 03 篇 |
| 蘋果自研、聯發科進攻、地緣政治——風險有多大？估值貴不貴？ | 第 04 篇 |

每個問題單獨回答都會得出"對的、但不完整的"結論。

## 一個預告：高通目前到底貴不貴？

留個鉤子。這些數字會在第 04 篇詳細拆解，先記住量級：

| 指標 | 資料 |
|------|------|
| 當前股價（2026年5月） | ~$209 |
| 市值 | ~2,170 億美元 |
| PE (TTM) | ~21倍 |
| FY2025 營收 | 443 億美元（+13.7%） |
| FY2025 自由現金流 | 128 億美元 |
| FCF 收益率 | ~6% |
| 5年平均 ROIC | **29.0%** |
| QTL 稅前利潤率 | **>63%** |
| 股東回報 | 200 億美元回購計劃 + 年化股息 $3.68/股 |
| 汽車設計贏單管線 | **450 億美元** |
| 蘋果基帶收入（將在 2027 年歸零） | **~57-59 億美元/年** |
| 中國市場收入佔比 | **~60%** |

把這些放一起，你看到的高通是這樣的：

- 一家 ROIC 近 30%、有"免費安全墊"（專利授權）的半導體公司
- 估值 21 倍本益比，在同業中偏低（博通約 35 倍、德州儀器約 32 倍）
- 汽車晶片增長迅猛，450 億設計贏單提供了高可見度
- 但最大客戶蘋果正在脫鉤，60% 收入來自地緣政治敏感的中國市場

**同一家公司，看多派說"21 倍本益比買一台專利印鈔機+汽車增長引擎"，看空派說"蘋果要走、中國要出事、手機見頂了"。**

這正是接下來三篇要拆清楚的。
