# PayPal：8倍本益比的支付巨頭，是撿便宜還是接飛刀？

> 閱讀時間約 15 分鐘

---

## 一組讓你重新審視PayPal的數字

先看一組資料，不做任何評價：

| 指標 | 資料 |
|------|------|
| 年收入 | **332億美元** |
| 年自由現金流 | **56億美元** |
| 淨現金 | **150億美元** |
| 活躍賬戶 | **4.39億** |
| 年處理支付 | **1.79萬億美元** |
| 當前市值 | **364億美元** |
| 本益比 | **8.7倍** |
| 股價距歷史高點 | **跌了87%** |

一家年賺56億美元現金的公司，市場給364億定價——相當於只用不到7年的現金流就能回本。它的市值幾乎等於一年的收入。

**如果這是一家沒人聽過的小公司，這些數字足以讓任何價值投資者興奮。**

但這是PayPal——一家從310美元跌到41美元、兩年半換了三個CEO的公司。便宜從來不是沒有原因的。

---

## 從310美元到41美元：發生了什麼

要理解今天的PayPal，得先搞懂它是怎麼從天上摔到地上的。

**2021年7月，股價310美元。** 疫情把全球消費者推向線上，PayPal是最大受益者。市場認為「電商永遠改變了世界」，給了PayPal超過60倍的本益比。

**然後泡沫破了。**

疫情紅利消退，電商增速回歸常態。更致命的是，蘋果支付、Stripe、Shopify開始從各個方向蠶食PayPal的領地。PayPal同時在追逐加密貨幣、先買後付、超級應用——什麼都想做，什麼都沒做好。

利潤率從22.8%一路跌到7.8%。股價從310跌到50。

**2023年底觸底，新CEO Alex Chriss上任，推動「利潤率優先」改革。** 2024年出現改善跡象，股價反彈到80。市場一度以為轉機來了。

**2025年Q4，品牌結賬增速驟降到1%。股價暴跌10%。**

**2026年2月，董事會突然換CEO，換來的是一個賣印表機的。** 股價再次崩塌，跌至41美元。

這不是「觸底反彈」的故事。這是「觸底→反彈→發現問題沒解決→二次探底」的故事。

---

## PayPal到底賺誰的錢

PayPal的收入來自四個地方，理解它們的差異至關重要：

### 品牌結賬：利潤最厚，但長不動了

當你在網上購物時看到「PayPal按鈕」並點選支付——這就是品牌結賬。PayPal每筆抽成約3.5%，利潤率極高。

問題是：**這個按鈕越來越不被需要了。**

蘋果支付用面容識別一秒完成支付，不需要跳轉到PayPal頁面、不需要登入、不需要記密碼。對年輕人來說，PayPal按鈕是個「多餘的步驟」。

資料已經反映了這一點：品牌結賬增速從6%降到1-2%。美國蘋果支付使用者（9,050萬）和PayPal使用者（9,210萬）的差距幾乎為零。

**這是PayPal最高利潤率的核心業務，也是跌得最痛的一塊。**

### Braintree：增長最快，但幾乎不賺錢

你在網上用信用卡付款時，可能根本沒意識到背後是PayPal在處理——這就是Braintree。它是一個「看不見的後端」，幫商戶處理支付。

Braintree增長很快（交易量+11%），但為了搶客戶，價格壓得很低。**這是一個「賠本賺吆喝」的生意**——增長越快，整體抽成率越低。

**核心矛盾就在這裡：增長最快的業務利潤最低，利潤最高的業務長不動。**

### Venmo：年輕人的錢包

Venmo是美國最流行的P2P支付應用——朋友之間分賬、AA付款、轉賬。月活超1億，在年輕人中品牌認知度極高。

好訊息是Venmo終於開始賺錢了。2025年收入17億美元（+20%），簽帳金融卡增長60%，商戶支付增長34%。

壞訊息是17億隻佔PayPal總收入的5%。**Venmo是一個「好故事」但還不是「大生意」。**

### 新業務：廣告、穩定幣、AI

PayPal正在用25年積累的交易資料做廣告——和Meta/谷歌不同，PayPal知道你**真的買了什麼**而不只是看了什麼。

PYUSD穩定幣市值從6億飆升到41億，擴充套件至70個市場。

AI被用於反欺詐（每季度攔截5億美元欺詐）和智慧結賬。

**這些都是方向正確的嘗試，但目前對收入的貢獻幾乎可以忽略。**

## 護城河：還在，但在變薄

### 仍然存在的優勢

1. **全球最大的線上支付網路** — 4.39億賬戶+3,500萬商戶的雙邊網路不是一夜能建成的
2. **信任品牌** — 在陌生網站上看到PayPal，消費者更願意付款。這種信任花了20年建立
3. **風控能力** — 500個資料點、5億美元季度反欺詐，這對商戶有真實價值
4. **56億美元年現金流** — 再怎麼唱衰，這台機器每年還在穩定地印鈔票

### 正在被侵蝕的部分

1. **定價權在下降** — 抽成率連年走低（1.73%），業務組合越來越偏向低利潤
2. **品牌結賬被繞過** — 蘋果支付用作業系統級優勢讓PayPal按鈕變得「可有可無」
3. **使用者基座在縮** — 活躍賬戶從峰值下降，年輕人不再把PayPal當首選
4. **管理層不穩定** — 兩年半三個CEO，戰略方向不確定

## 便宜還是陷阱：一個關鍵問題

市場給PayPal 8.7倍本益比，只有歷史中位數的五分之一。這意味著市場在定價以下幾種可能之一：

**可能A：市場錯了，PayPal被嚴重低估**
- 332億收入、56億現金流的公司不值得只有364億市值
- 每年60億回購，3年後流通股減少50%，每股盈餘可能翻倍
- Venmo和廣告是免費贈送的期權
- Michael Burry在49美元建倉，說明聰明錢看到了價值

**可能B：市場對了，PayPal在緩慢衰退**
- 品牌結賬份額丟失是結構性的，不可逆
- 蘋果支付將在5-10年內取代大部分線上PayPal結賬
- 支付正在變成基礎設施，沒人願意為基礎設施付溢價
- 回購跑不贏業務的萎縮

**決定你站哪一邊的，就一個問題：品牌結賬能不能止跌？**

如果能——8.7倍PE是10年一遇的便宜。
如果不能——低PE不是安全墊，而是衰退預告。

## 三種情景，三種命運

| 情景 | 假設 | 合理估值 | 上行/下行 |
|------|------|---------|----------|
| **樂觀** | 品牌結賬恢復5%增長 + Venmo加速 + 廣告貢獻 | 700-800億美元 | **股價翻倍** |
| **中性** | 品牌結賬低個位數增長 + 利潤率穩定 + 回購驅動每股盈餘 | 500-600億美元 | **+30-50%** |
| **悲觀** | 品牌結賬持續丟份額 + 利潤率下行 + 商品化 | 300-350億美元 | **-15-25%** |

## 值得跟蹤的訊號

如果你在考慮是否投資PayPal，以下幾個訊號比任何分析師報告都重要：

**看多訊號：**
- 品牌結賬季度增速回到3%以上
- Fastlane一鍵結賬的商戶採用率加速
- Venmo季度收入增速保持20%+
- 新CEO提出清晰且可執行的戰略

**看空訊號：**
- 品牌結賬增速降至零或負增長
- 活躍賬戶跌破3.5億
- 營業利潤率連續下行
- 又一次換CEO

## 最後一段話

PayPal讓我想到一個比喻。

它像一座正在漏水的大壩。壩體本身——332億收入、56億現金流、150億淨現金——依然堅固。但裂縫——品牌結賬減速、管理層動盪、蘋果競爭——在擴大。

8.7倍本益比，是市場給出的修繕價格。

如果裂縫能堵住，這個價格極度便宜。如果堵不住，大壩終將被侵蝕。

Michael Burry在49美元建倉說明他賭前者。市場跌到41美元說明多數人賭後者。

你賭哪一邊？
