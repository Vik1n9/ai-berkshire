# PayPal vs 螞蟻集團：兩大支付巨頭的異同

> 分析日期：2026年6月8日
> 本文不構成任何投資建議。

---

## 相同點

| 維度 | 共同特徵 |
|------|---------|
| 起源 | 都從第三方支付起家（PayPal→eBay生態，支付寶→淘寶生態） |
| 核心功能 | 都解決了"買家不信任賣家"的信任問題——中間託管資金 |
| 規模 | 都是各自市場的支付霸主（PayPal全球線上43%，支付寶中國36%） |
| 超級應用野心 | 都試圖從支付延伸到金融（信貸、理財、保險）+ 生活服務 |
| 監管壓力 | 都經歷過嚴厲監管打擊（螞蟻IPO叫停、PayPal被英國FCA反壟斷調查） |
| 估值壓縮 | 都從巔峰大幅縮水（螞蟻從2萬億降至6000億估值，PayPal從3000億降至364億市值） |

---

## 核心差異

| 維度 | PayPal | 螞蟻集團 |
|------|--------|---------|
| **市場** | 全球化（200+國家），核心在美國/歐洲 | 中國為主，國際化剛起步 |
| **底層架構** | 建在Visa/Mastercard卡組織之上——**"二房東"** | 直連銀行賬戶，繞過卡組織——**"房東"** |
| **抽成結構** | 收3.49%但要分一半給卡組織和髮卡行，自己留約1.5% | 費率更低（約0.6%），但直連銀行不用分給卡組織 |
| **使用者粘性來源** | 品牌信任+買家保護（功能層面） | 社交關係+生活場景+小程式生態（生態層面） |
| **最大威脅** | 蘋果支付（作業系統級替代） | 微信支付（社交場景侵蝕，份額已反超至60%） |
| **金融業務** | 輕度涉足（先買後付、信貸），主動收縮 | 深度佈局（花唄、借唄、餘額寶、保險），被監管強制收縮 |
| **盈利能力** | 淨利率約16%，自由現金流56億美元 | 2025年淨利潤153億元（同比-60%），研發投入290億創新高 |
| **上市狀態** | 納斯達克上市，市值364億美元 | 未上市，IPO擱淺至今 |
| **AI戰略** | 風控+廣告+Fastlane結賬（應用層） | 螞蟻阿福+百靈大模型（基礎設施層） |

## 收入漏斗對比

### PayPal

```
總支付量 1.79萬億美元
    │ × 抽成率1.73%
營收 332億美元
    │ - 卡組織/髮卡行/欺詐成本 177億
交易利潤額 155億美元
    │ - 運營費用 94億
營業利潤 61億美元
    │ - 稅
淨利潤 52億美元 → 自由現金流 56億美元
```

轉化率：自由現金流/總支付量 = **0.31%**

### 螞蟻集團（推算，資料有限）

```
總支付量 約118萬億元（2023年支付寶資料推算）
    │ × 抽成率約0.06%（遠低於PayPal）
營收 約1,900億元（2023年估）
    │ - 運營成本
淨利潤 153億元（2025年，受整改和AI投入影響大幅下降）
```

轉化率：淨利潤/總支付量 = **約0.001%**

### 關鍵差異

**PayPal的抽成率（1.73%）是螞蟻的近30倍**——這不是因為PayPal更能賺錢，而是因為兩者的底層架構完全不同：

- PayPal建在Visa/Mastercard之上，卡組織費率本身就高（美國信用卡交換費約2%），PayPal在此基礎上再加一層
- 螞蟻直連銀行賬戶，沒有卡組織中間商，整個支付鏈條的成本更低
- 中國支付市場被微信和支付寶培養成「近乎免費」的使用習慣，抽成空間極其有限

---

## 最本質的差異

**PayPal是卡組織生態的"增值層"，螞蟻是銀行體系的"替代層"。**

PayPal的每筆交易都要經過Visa/Mastercard的鐵路，交"過路費"。它賺的是在卡組織之上提供的"信任+便利"溢價。問題是：當蘋果支付用作業系統原生能力提供同等信任和更好便利時，這層溢價就被擠壓。

螞蟻（支付寶）直接連線使用者銀行賬戶，不依賴卡組織——中國沒有Visa/Mastercard這層中間商。它的商業模式更像是"自建鐵路"而非"鐵路上的服務商"。但這也意味著它直接觸碰了銀行和監管的核心利益，所以被整改得更狠。

**PayPal的天花板是卡組織讓它賺多少，螞蟻的天花板是監管讓它做多少。**
