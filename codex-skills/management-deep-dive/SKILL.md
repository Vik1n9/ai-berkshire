## 設計理念

大多數投資分析對管理層的評估停留在表面：履歷、持股比例、薪酬。但巴菲特花大量時間**和管理層吃飯聊天**，李錄說**他投資的本質是投人**，段永平說**買股票就是買人**。

本Skill是 `/investment-research` 第五步管理層評估的**深化版**。當標準投資研究中管理層評分不確定（★★★或以下）、或管理層是核心投資邏輯時，使用本Skill做縱深研究。

AI無法和管理層吃飯，但可以透過公開資訊做到：
- **追蹤管理層的話與做是否一致**（承諾vs兌現）
- **分析每一筆重大資本配置決策的回報**
- **從困難時期的決策中推斷品格**
- **透過員工/商家/客戶的反饋側面驗證**

### 第二步：CEO能力圈評估

#### 2.1 戰略眼光

搜尋CEO過去5年的公開發言（股東信、電話會、採訪、社交媒體），提取其對以下問題的判斷：

| 時間 | CEO的判斷/預測 | 實際結果 | 準確度 |
|------|--------------|---------|:------:|
| | "我們認為X市場會..." | X市場實際... | ✅/❌ |
| | "未來3年我們的重點是..." | 實際執行... | ✅/❌ |

**關鍵問題**：
- CEO有沒有做過超前於市場的正確判斷？
- CEO有沒有在大家都看好的時候保持冷靜？
- CEO對行業趨勢的理解是跟隨市場還是獨立思考？

### 第三步：誠信度評估（最重要）

**巴菲特**："我們尋找三種品質：正直、智慧和精力。如果沒有第一種，後兩種會害死你。"

#### 3.1 承諾vs兌現追蹤

從過去3年的財報電話會、股東信、公開採訪中，提取管理層做過的**具體承諾**：

| # | 時間 | 承諾內容 | 承諾場合 | 兌現情況 | 評價 |
|---|------|---------|---------|---------|------|
| 1 | | "我們將在2025年實現X業務盈利" | 2024年報電話會 | | ✅/⚠️/❌ |
| 2 | | "我們計劃回購$X億" | 2024年股東信 | | ✅/⚠️/❌ |

**兌現率統計**：

| 承諾兌現率 | 評價 |
|:---------:|------|
| >80% | 優秀——說到做到 |
| 60-80% | 合格——大方向對但執行有偏差 |
| 40-60% | 令人擔憂——承諾過多交付不足 |
| <40% | 嚴重問題——不可信賴 |

#### 3.2 困難時期的表現

搜尋公司歷史上遭遇的重大危機/困難（股價暴跌、業績miss、監管衝擊、競爭加劇），分析管理層的應對：

| 危機事件 | 時間 | 管理層反應 | 事後回看的評價 |
|---------|------|-----------|-------------|

**關注**：
- 是主動溝通還是躲避？
- 是歸因內部還是甩鍋外部？
- 是趁機做困難但正確的事，還是選擇短期討好市場？

#### 3.3 對利益相關方的態度

| 利益相關方 | 管理層態度 | 證據 | 評價 |
|-----------|-----------|------|------|
| 股東 | 尊重/忽視/利用 | | |
| 員工 | 善待/壓榨/漠視 | | |
| 客戶/使用者 | 以客戶為中心/短期榨取 | | |
| 商家/供應商 | 公平合作/極端壓價 | | |
| 監管/社會 | 合規配合/打擦邊球 | | |

**李錄**："對利益相關方的態度決定了企業的長期生命力。短期壓榨能提升效率，但長期會損害生態。"

#### 段永平的"買人"標準

> 回答以下三個問題：
> 1. **這個人是否正直？**（誠實、不佔股東便宜）
> 2. **這個人是否有能力？**（戰略眼光+執行力+資本配置）
> 3. **你願意把錢交給這個人管10年嗎？**
>
> 三個都是"是" = ★★★★★（5分）
> 前兩個是"是" = ★★★★（4分）
> 只有第一個是"是" = ★★★（3分）
> 第一個不是"是" = ★（1分，不投）

## 關鍵原則

- **誠信是一票否決項** — 能力不足可以學習，品格有問題無法修復
- **看行為不看言辭** — 管理層說什麼不重要，做了什麼才重要
- **在困難中看真相** — 順風時誰都是好CEO，逆風時才見真功夫
- **資本配置是終極考試** — 賺錢容易，把賺到的錢配置好難
- **不要愛上管理層** — 保持客觀，即使是你欣賞的人也可能犯大錯
