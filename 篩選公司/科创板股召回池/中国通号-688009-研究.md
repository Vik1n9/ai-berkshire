# 中國通號（688009.SH）深度投資研究

> 研究日期：2026年5月17日
> 資料截止：2025年年報 + 2026Q1季報

---

## 一、商業模式與競爭壁壘

### 1.1 主營業務拆解（2024年收入結構）

| 業務板塊 | 營收（億元） | 佔比 | 同比增速 | 毛利率 |
|---------|------------|------|---------|--------|
| 鐵路領域 | 188.90 | 58% | -1.86% | 29.95% |
| 城軌領域 | 82.20 | 25% | -3.58% | 33.47% |
| 海外業務 | 24.41 | 7% | +64.20% | 24.61% |
| 工程總承包 | 26.89 | 8% | -64.96% | 11.98% |

來源：[2024年報點評-新浪財經](https://finance.sina.com.cn/roll/2025-03-31/doc-inerqmnh3624987.shtml)

**商業模式本質**：覆蓋軌道交通控制系統"設計諮詢→裝置製造→系統整合→運維服務"全產業鏈。鐵路+城軌核心業務佔比83%，是典型的"國家基建配套型"企業。

### 1.2 行業地位與市佔率

- **高鐵控制系統**：國內市佔率約60%，絕對龍頭
- **城軌訊號系統**：國內市佔率約48.5%（含子公司卡斯柯、通號城軌），第一名
- **全球排名**：全球最大軌道交通控制系統供應商之一
- 擁有5500+項註冊專利，列控核心技術100%自主化

來源：[騰訊新聞-交通強國報告](https://news.qq.com/rain/a/20230428A01MRE00)

### 1.3 護城河分析

**強護城河因素**：
- **技術壁壘極高**：軌道交通訊號系統關乎行車安全，認證週期長（3-5年），新玩家幾乎無法進入
- **客戶粘性強**：國鐵集團/地鐵公司是核心客戶，替換成本極高
- **標準制定者**：參與國內軌道交通訊號技術標準制定
- **規模效應**：研發投入超20億/年，中小企業無力跟進

**護城河弱化因素**：
- 國企體制，市場化激勵不足
- 城軌領域競爭者逐漸增多（交控科技、和利時等蠶食份額）
- 依賴政府鐵路/城軌投資計劃，議價權有限

來源：[2025年報簡析-騰訊新聞](https://news.qq.com/rain/a/20260326A025GZ00)、[2024年報簡析-騰訊新聞](https://news.qq.com/rain/a/20250330A01N5Q00)

**犀利點評**：ROE年均約9%（歷史），近兩年持續下滑至7%左右。作為重資產國企，資本回報率平庸。毛利率29%看似不錯，但扣除工程總承包後核心業務毛利率30-33%，淨利率僅12%——管理費用率7.3%+研發費用率5.9%吃掉大量利潤。

| OCF/淨利潤 | 150% | 負值 |
| 應收賬款/淨利潤 | 771.83% | 678.21% |
| 收現比（5年均值） | 0.94 | — |

來源：[2024年報-新浪財經](https://finance.sina.com.cn/roll/2025-03-31/doc-inerqmnh3624987.shtml)、[2025年報簡析](https://news.qq.com/rain/a/20260326A025GZ00)

**犀利點評**：現金流是這家公司最大的硬傷。2025年經營現金流轉負，應收賬款長期高企（接近淨利潤的7倍）。5年平均收現比0.94，說明每賺100元賬面利潤只收回94元現金。作為一家"賺紙面利潤"的企業，自由現金流長期不理想，分紅能力受限。核心原因是客戶（國鐵集團、各地鐵公司）回款週期長，屬於典型的"大甲方小乙方"格局。

## 三、成長性

### 3.1 收入增長驅動力

1. **鐵路存量改造市場**：全國超1萬公里高鐵進入大修改造期，訊號系統升級是剛需
2. **海外市場爆發**：2024年海外營收+64%，2025H1海外營收+55.9%，已成為核心增長極
3. **低空經濟新賽道**：構建"低空空域管控+無人機制造+服務運營"佈局，2025H1簽約0.3億元（體量極小）
4. **城軌後市場**：既有線路維護升級需求

### 3.2 在手訂單情況

- **截至2025年6月底**：在手訂單1175.63億元
- 相當於2024年營收的**3.6倍**，保障至少3年正常經營
- 2025年上半年新籤合同175.68億元

來源：[新浪財經-在手訂單報道](https://finance.sina.com.cn/roll/2025-09-29/doc-infscaav8404282.shtml)

### 3.3 海外拓展進度

- 2024年海外營收24.41億元（+64.2%），佔比7%
- 2025H1海外營收9.44億元（+55.9%）
- 主要覆蓋"一帶一路"沿線國家鐵路訊號系統出口
- 海外毛利率24.61%，低於國內，但增速快、空間大

來源：[銀河證券研報](https://bigquant.com/square/paper/9e7306d6-6b11-4fe3-8ab2-0a607b2b2c6d)

**犀利點評**：增長故事有邏輯但缺乏爆發力。鐵路改造是"維持型"而非"增長型"需求；低空經濟簽約0.3億對346億營收而言微不足道，至少3-5年內無法貢獻實質收入；海外業務增速亮眼但基數小，且毛利率低於國內。整體判斷：低個位數增長的成熟企業，非高成長股。

## 四、估值與風險

### 4.1 當前估值水平

| 指標 | 數值 |
|------|------|
| 最新股價 | ~5.07元（2026年5月） |
| 總股本 | ~105.9億股 |
| 總市值 | ~537億元 |
| 2025年EPS | 0.34元 |
| PE TTM | ~14.9倍 |
| PB | ~1.13倍 |
| 股息率 | ~2.5%（估算） |

來源：[騰訊證券](https://gu.qq.com/sh688009)、[東方財富](https://data.eastmoney.com/stockdata/688009.html)

**估值判斷**：15倍PE對應7%的ROE和個位數增長，談不上便宜。PB僅1.1倍反映市場對其資產質量（大量應收賬款）的擔憂。與同類國企（中國中車PE 20x+）相比估值略低，但盈利質量也更差。

### 4.2 主要風險點

1. **應收賬款風險**：應收賬款佔淨利潤的7倍，若出現壞賬集中暴露將嚴重侵蝕利潤
2. **鐵路投資週期風險**：國家鐵路投資若大幅下滑，公司營收將承壓
3. **現金流惡化風險**：2025年經營現金流已轉負，若持續惡化將影響分紅和再投資能力
4. **城軌競爭加劇**：交控科技等競爭者持續蠶食城軌份額
5. **國企治理效率**：ROE持續走低反映資本配置效率不高
6. **2026Q1業績下滑**：營收-3.5%、淨利潤-7.2%，增長動能減弱

### 4.3 投資結論

| 維度 | 評分 | 說明 |
|------|------|------|
| 商業模式 | ★★★★ | 軌交訊號龍頭，壁壘高、地位穩 |
| 盈利能力 | ★★☆ | ROE僅7%，資本回報平庸 |
| 現金流質量 | ★★ | 長期"紙面利潤"，應收賬款巨大 |
| 成長性 | ★★☆ | 海外有亮點但整體低增長 |
| 估值吸引力 | ★★★ | 15x PE不貴但也不便宜 |
| **綜合評分** | **★★☆** | **2.5星/5星** |

### 最終判斷

中國通號是一家"好生意、差公司"的典型案例。軌交訊號系統本身壁壘極高、市佔率穩固，但國企體制下資本配置效率低下、現金流質量差、增長乏力。15倍PE對應7%的ROE和個位數增長，沒有明顯的安全邊際。

**不適合作為核心持倉**。除非出現以下情況才值得重新評估：
- PE降至10倍以下（股價3.4元）
- 海外業務佔比突破20%且毛利率改善
- 現金流質量實質性改善（收現比持續>1）
