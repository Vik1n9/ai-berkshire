## 核心發現

1. **半導體裝置是最優質賽道**——7家透過，"賣鏟人"模式經去劣檢驗最紮實
2. **高毛利是好公司的共性**——透過公司毛利率中位數~45%，對比全部100家中位數~25%
3. **科創板整體盈利質量堪憂**——56%的公司因虧損被淘汰，"科技成長"≠"好生意"
4. **值得長期跟蹤的邊界公司**——海光資訊、拓荊科技、中國通號等18家需等待更多財務資料驗證
