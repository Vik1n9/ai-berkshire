## 1. 恆瑞醫藥（600276）

**市值**：3680億 | **PE(TTM)**：48倍 | **2026E PE**：39倍

### 核心投資邏輯

A股創新藥絕對龍頭，創新藥收入163億佔比58%，轉型成功。研發管線極豐富，覆蓋腫瘤、自免、代謝等熱門靶點，未來3年有多個重磅品種進入收穫期。

### 護城河分析

研發平台型公司，同時推進60+臨床專案，管線深度和廣度國內無人能及。銷售網路覆蓋全國，學術推廣能力強。但創新藥行業競爭激烈，單品護城河有限，依賴持續創新。管理層研發投入決心堅定，孫飄揚迴歸後戰略清晰。

### 估值判斷

48倍PE對創新藥龍頭而言偏貴，市場給予了確定性溢價。除非有重磅品種超預期，否則當前價格缺乏安全邊際。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河 | ★★★★ |
| 管理層 | ★★★★★ |
| 估值 | ★★ |
| 綜合 | ★★★★ |

**結論**：好公司但估值偏貴，適合跟蹤等待回撥，重點關注GLP-1等大品種進展。

## 2. 邁瑞醫療（300760）

**市值**：2160億 | **PE(TTM)**：27倍 | **2026E PE**：21倍

### 核心投資邏輯

國產醫療器械絕對龍頭，全球排名前20。產品覆蓋生命資訊與支援、體外診斷、醫學影像三大領域。2025年受反腐影響營收-9.4%，但海外+7.4%證明全球競爭力。業績低谷+合理估值，是經典的"好公司遇到短期困難"。

### 護城河分析

產品線極寬：三大業務板塊互相協同，打包銷售能力強。技術壁壘：高階監護儀、超聲等產品效能接近GPS（GE/飛利浦/西門子）。全球渠道：覆蓋190+國家，品牌認知度持續提升。醫療器械粘性極強，醫院一旦採購很少切換。研發費率10%+保證持續創新。

### 估值判斷

27倍PE處於歷史低位區間，若2026年業績恢復增長則21倍PE極具吸引力。反腐影響是一次性的，不改變長期邏輯。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★★ |
| 護城河 | ★★★★★ |
| 管理層 | ★★★★★ |
| 估值 | ★★★★ |
| 綜合 | ★★★★★ |

**結論**：強烈推薦深研，好公司+好價格+短期困境創造買入機會，是醫藥板塊最優選擇。

## 3. 藥明康德（603259）

**市值**：3000億 | **PE(TTM)**：16倍

### 核心投資邏輯

全球CXO龍頭，在手訂單599億（+41%）證明客戶需求強勁。被美國生物安全法案壓制估值，16倍PE對應全球藥企研發外包不可逆趨勢，是政策恐慌創造的定價錯誤。

### 護城河分析

一體化CRDMO平台全球獨一無二，從藥物發現到商業化生產一站式服務。客戶粘性極強：藥物一旦進入臨床，切換CXO供應商意味著重新申報，成本極高。全球前20大藥企中19家是其客戶。化學合成能力全球頂尖，產能規模無人能及。風險在於地緣政治不確定性。

### 估值判斷

16倍PE是歷史絕對底部，即使考慮地緣風險折價也明顯低估。在手訂單高增長說明實際業務未受影響，市場過度反應政策風險。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★★ |
| 護城河 | ★★★★ |
| 管理層 | ★★★★ |
| 估值 | ★★★★★ |
| 綜合 | ★★★★★ |

**結論**：強烈推薦深研，政策恐慌壓制的估值極具吸引力，核心跟蹤在手訂單執行和美國政策落地情況。

## 4. 片仔癀（600436）

**市值**：1030億 | **PE(TTM)**：45倍

### 核心投資邏輯

國家絕密配方+天然麝香獨佔權，護城河深度可能是A股之最。品牌溢價極強，持續提價能力驗證定價權。但2025年首次出現雙降（營收和利潤），需判斷是週期性還是結構性問題。

### 護城河分析

獨一無二的護城河：國家絕密配方不可複製，天然麝香來源受國家管控僅供片仔癀使用。品牌歷史超500年，在東南亞華人圈有"藥中茅台"之稱。產品無替代品，需求剛性極強。唯一風險是天然麝香供給瓶頸限制產能擴張。

### 估值判斷

45倍PE對增速放緩的消費品而言偏貴。護城河雖深但增長空間有限，需要更大的安全邊際。理想買入區間PE 30倍以下。

| 管理層 | ★★★ |

## 5. 愛爾眼科（300015）

**市值**：1000億 | **PE(TTM)**：31倍

### 核心投資邏輯

全球最大眼科連鎖醫療機構，分級連鎖模式實現快速擴張。增速放緩至個位數但估值已回落至歷史低位，近視防控和老齡化白內障需求長期確定。

### 護城河分析

網路效應：全球800+家醫院/診所構成的分級連鎖網路難以複製。品牌認知度在眼科領域絕對領先，患者選擇眼科手術首先想到愛爾。醫生培養體系成熟，人才壁壘明顯。但連鎖模式面臨管理半徑問題，商譽風險值得關注。

31倍PE處於歷史低位，若增速能恢復至15%+則具備配置價值。需觀察2026年門診量恢復情況。

## 6. 長春高新（000661）

**市值**：524億 | **利潤暴跌94%**

### 核心投資邏輯

生長激素絕對龍頭（金賽藥業），遭醫保降價衝擊後利潤暴跌94%。屬於高風險高收益標的：若生長激素需求不可逆萎縮則價值歸零，若只是短期陣痛則當前極度低估。

### 護城河分析

生長激素技術壁壘高，水針劑型和長效劑型國內領先。但醫保降價直接打擊核心盈利模式，護城河被政策削弱。適應症拓展（成人生長激素缺乏、輔助生殖等）是潛在增量。金賽藥業品牌和渠道仍有優勢，但政策風險懸而未決。

### 估值判斷

利潤暴跌導致PE失真，需用正常化利潤估算。若恢復至50億利潤水平對應10倍PE極具吸引力，但恢復的確定性存疑。

| 護城河 | ★★★ |
| 管理層 | ★★★ |

## 7. 華蘭生物（002007）

**市值**：270億 | **PE(TTM)**：29倍 | **ROE**：8.2%

### 核心投資邏輯

血製品行業壁壘極高（採漿站牌照管制），供給側天然受限。但ROE僅8.2%，資本效率偏低，說明管理層對資本的運用能力一般。

### 護城河分析

血製品行業進入壁壘極高：採漿站牌照審批嚴格，新進入者幾乎不可能。原料（血漿）稀缺性保證供不應求格局。但公司在行業內並非絕對龍頭（天壇生物更大），產品結構和噸漿利潤有提升空間。疫苗業務波動較大。

### 估值判斷

29倍PE對應8.2%的ROE價效比不高。除非ROE能提升至15%+，否則不具備投資吸引力。

| 護城河 | ★★★★ |
| 管理層 | ★★★ |
| 估值 | ★★★ |
