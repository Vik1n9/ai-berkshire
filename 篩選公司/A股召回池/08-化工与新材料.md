## 1. 萬華化學（600309.SH）

**市值**：約2800億元 | **PE(2026E)**：約9倍 | **股息率**：約3%

### 核心投資邏輯

全球MDI龍頭，市佔率約30%。MDI全球僅5家供應商（萬華、巴斯夫、科思創、亨斯邁、陶氏），寡頭格局極為穩固。2026年是產能釋放大年（蓬萊基地、眉山基地投產），預計淨利潤184億元（同比+28%）。萬華的核心競爭力在於持續的技術突破和成本控制——單噸MDI成本全球最低。

### 護城河分析

技術+資本+環保三重壁壘構成極強護城河。技術壁壘：MDI生產工藝極其複雜（涉及光氣化反應，安全要求極高），全球50年僅5家公司掌握；萬華是唯一透過自主研發掌握全套技術的中國企業。資本壁壘：新建MDI裝置投資50-100億元，投產週期3-5年。環保壁壘：光氣劇毒，環評審批幾乎不可能透過新增產能。但另一方面，MDI下游（冰箱保溫、汽車、建築）增速有限（5-7%），萬華的高增長更多來自新材料板塊（POE、PMMA、檸檬醛等），這些新業務的競爭格局尚未完全穩固。

### 估值判斷

便宜。PE(2026E) 9倍，對於一家ROE 15%+、增速20%+、具有全球寡頭地位的化工龍頭而言，估值極具吸引力。歷史PE中樞在15-20倍，當前折價顯著。市場擔憂的是化工週期下行和房地產需求疲軟，但萬華透過產品多元化和全球化正在弱化週期性。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★★ |
| 護城河強度 | ★★★★★ |
| 管理層質量 | ★★★★★ |
| 估值吸引力 | ★★★★★ |
| **綜合評分** | **★★★★★** |

**結論**：強烈推薦深度研究。全球化工領域極少數同時具備"極強護城河+優秀管理層+低估值"的公司。廖明夫（董事長）團隊的技術創新能力和戰略視野在中國化工行業中無出其右。

## 2. 華魯恆升（600426.SH）

**市值**：約700億元 | **PE(2026E)**：約9-10倍 | **股息率**：約3.5%

### 核心投資邏輯

煤化工領域的"低成本之王"。核心競爭力在於柔性生產平台——同一套裝置可以根據市場價格靈活切換生產尿素、DMF、己二酸、醋酸等多種產品，始終生產利潤率最高的產品。預計2026年淨利潤68億元（同比+17%）。荊州基地投產將開啟新增長空間。

### 護城河分析

成本優勢+柔性生產構成差異化壁壘。華魯恆升的煤氣化裝置效率全行業領先，單位能耗比同行低15-20%，這是多年工藝最佳化積累的結果。柔性聯產模式使公司在任何化工產品價格組合下都能保持盈利，抗週期能力遠超單一產品企業。但另一方面，煤化工行業整體面臨碳中和政策約束，長期需要向綠色化工轉型；且公司主要產品（尿素、DMF、己二酸）壁壘不如萬華的MDI高，新進入者持續出現。

### 估值判斷

便宜。PE 9-10倍，對應ROE 15%+和穩定的分紅。在化工行業中，華魯恆升的低成本優勢使其即使在週期底部也能保持盈利（歷史上從未虧損），這種確定性值得溢價但市場並未給予。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★★ |
| 估值吸引力 | ★★★★★ |
| **綜合評分** | **★★★★★** |

**結論**：值得深度研究。煤化工領域的"成本殺手"，柔性生產模式是獨特競爭優勢。估值極具安全邊際，管理層執行力突出。

## 3. 北新建材（000786.SZ）

**市值**：約470億元 | **PE(TTM)**：約14.5倍 | **ROE**：約20%+

### 核心投資邏輯

石膏板絕對龍頭，國內市佔率超60%。石膏板行業的競爭格局極好——北新建材一家獨大，第二名泰山石膏（也歸屬中國建材集團）市佔率僅20%+，形成事實上的壟斷。產品單價低（每平米幾塊錢）但運輸半徑有限（經濟運輸半徑約300公里），因此天然形成區域壟斷格局。ROE常年20%+，現金流優秀。

### 護城河分析

規模+運輸半徑構成極強壁壘。北新建材在全國佈局了100+條產線，每條產線覆蓋周邊300公里市場，新進入者即使有技術也無法在已有產線的區域盈利（運輸成本會吃掉利潤）。品牌"龍牌"在裝修市場認知度極高。但另一方面，石膏板需求與房地產高度相關（新房裝修+存量翻新），在地產持續下行期面臨需求壓力；公司正透過防水、塗料等品類擴張尋求第二曲線，但效果尚待驗證。

### 估值判斷

合理偏低。PE 14.5倍，對應20%+ ROE和60%+市佔率的壟斷地位，估值具有吸引力。地產觸底將帶來估值修復催化。歷史PE中樞在18-22倍，當前折價約30%。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★★ |
| 護城河強度 | ★★★★★ |
| 管理層質量 | ★★★★ |
| 估值吸引力 | ★★★★ |
| **綜合評分** | **★★★★★** |

**結論**：值得深度研究。石膏板壟斷地位+運輸半徑壁壘是教科書級別的護城河。估值合理偏低，地產見底將是催化劑。

## 4. 中國巨石（600176.SH）

**市值**：約1100億元 | **PE(2026E)**：約16-17倍

### 核心投資邏輯

全球玻纖龍頭，產能全球第一。玻纖行業形成中國巨石、泰山玻纖、重慶國際"三巨頭"寡頭格局，CR3超70%。下游受益於風電葉片、汽車輕量化、AI伺服器PCB等高增長領域。玻纖價格2025年底已觸底回升，2026年處於景氣上行週期。

### 護城河分析

規模+技術+資本構成壁壘。玻纖窯爐投資大（單條線10億+）、建設週期長（2年+）、技術門檻高（窯爐設計和玻璃配方是核心know-how）。中國巨石在E9超細紗、高模量玻纖等高階產品上技術領先。寡頭格局下價格競爭有底線。但另一方面，玻纖行業具有明顯週期性，產能擴張與需求波動導致價格大幅波動；且年內已漲90%，短期透支了景氣回升預期。

### 估值判斷

合理但短期偏貴。PE(2026E) 16-17倍對應景氣上行期的利潤，如果2027年行業供給過剩、利潤回落，實際PE將膨脹。年內漲幅90%意味著短期獲利盤豐厚，回撥風險大。好公司，但需要等更好的價格。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★ |
| 估值吸引力 | ★★★ |
| **綜合評分** | **★★★★** |

**結論**：全球玻纖龍頭，寡頭格局好。但年內漲90%後短期偏貴，建議等回撥20-30%後再深度研究。

## 5. 新和成（002001.SZ）

**市值**：約1050億元 | **PE(2026E)**：約10-14倍

### 核心投資邏輯

全球維生素龍頭（VA全球前三、VE全球前二），同時向新材料（PPS、PPA等高效能工程塑膠）延伸。維生素行業寡頭格局穩定（全球僅5-6家主要供應商），供給端高度集中。2026年維生素價格處於景氣上行期（海外龍頭帝斯曼退出、巴斯夫產能事故），疊加新材料板塊放量，利潤預期樂觀。

### 護城河分析

技術+環保+寡頭格局構成壁壘。維生素合成路線複雜（VA合成步驟超20步），新進入者需要5-10年研發週期。環保審批趨嚴限制新增產能。全球寡頭格局（VA僅帝斯曼、巴斯夫、新和成、安迪蘇等）持續數十年。新材料板塊（PPS樹脂）打破日本東麗、DIC壟斷。但另一方面，維生素價格具有強週期性（歷史上漲跌幅可達3-5倍），新和成的利潤波動性較大；且新材料業務尚處於放量初期，能否複製維生素的成功仍需觀察。

### 估值判斷

便宜。PE 10-14倍（取決於維生素價格假設），對應全球寡頭地位和新材料第二曲線的成長性，估值具有吸引力。管理層（胡柏藩創始人家族）低調務實、長期投入研發，是典型的"隱形冠軍"經營風格。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★★ |
| 估值吸引力 | ★★★★ |
| **綜合評分** | **★★★★** |

**結論**：值得深度研究。全球維生素寡頭+新材料第二曲線，管理層低調務實。估值合理偏低，適合在維生素景氣底部佈局。

## 6. 金禾實業（002597.SZ）

**市值**：約180億元 | **PE(TTM)**：約15倍

### 核心投資邏輯

甜味劑（安賽蜜、三氯蔗糖）龍頭，受益於全球代糖消費趨勢。代糖在飲料、食品中的滲透率持續提升（健康意識+減糖政策雙重驅動）。金禾在安賽蜜領域全球市佔率超50%，三氯蔗糖全球前三。

### 護城河分析

細分領域的規模優勢和成本優勢。安賽蜜全球市佔率50%+，規模效應顯著。但另一方面，甜味劑行業整體壁壘偏低——三氯蔗糖的國內競爭者（金澳科技等）持續擴產，行業價格戰導致利潤率下滑；且赤蘚糖醇、阿洛酮糖等新一代甜味劑可能替代傳統產品，技術路線風險存在。護城河深度不如萬華、華魯恆升等企業。

### 估值判斷

合理。PE 15倍不貴也不便宜。考慮到競爭加劇和業績下滑趨勢，當前估值缺乏明顯安全邊際。需要觀察行業價格戰是否結束、公司新品（阿洛酮糖等）能否開啟新空間。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★ |
| 護城河強度 | ★★★ |
| 管理層質量 | ★★★ |
| 估值吸引力 | ★★★ |
| **綜合評分** | **★★★** |

**結論**：代糖趨勢好但競爭加劇、護城河偏淺。不優先深度研究。
