## 1. 招商銀行（600036.SH）

**市值**：約10100億元 | **PE(TTM)**：約6.3倍 | **股息率**：約5%

### 核心投資邏輯

零售銀行標杆，ROE常年13%+，不良率僅0.95%，撥備覆蓋率412%——資產質量在上市銀行中遙遙領先。招行的核心能力是低成本負債（活期佔比高）和零售客群經營能力（AUM超14萬億），這構成了持續的息差優勢。在利率下行環境中，招行的負債端優勢會被壓縮，但相對優勢仍然突出。

### 護城河分析

品牌+零售客群構成最深的銀行護城河。"招行"在零售金融領域的品牌心智極強，高淨值客戶粘性高、遷移成本大；金葵花及以上客戶貢獻了80%+的AUM，私人銀行業務規模行業第一。科技投入連續多年超百億，數字化能力領先同業。但另一方面，銀行業整體面臨息差收窄、利潤增速放緩的壓力，招行也不例外。

### 估值判斷

便宜。PE 6.3倍、PB約1.0倍，處於歷史較低分位。對比ROE 13%+的盈利能力，當前估值給予了充分的安全邊際。5%的股息率提供了持有的下限保護。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★★ |
| 管理層質量 | ★★★★ |
| 估值吸引力 | ★★★★ |
| **綜合評分** | **★★★★** |

**結論**：護城河最深的銀行，零售優勢構成長期競爭力。估值合理偏低，適合作為金融板塊核心持倉深度研究。

## 2. 中國平安（601318.SH）

**市值**：約11000億元 | **PE(TTM)**：約7-8倍 | **P/EV**：約0.64倍

### 核心投資邏輯

綜合金融龍頭，壽險改革進入收穫期。2023-2025年完成代理人隊伍清虛（從百萬級降至40萬+精英隊伍），人均產能大幅提升。NBV（新業務價值）連續恢復正增長，壽險基本面拐點確立。P/EV 0.64倍意味著市場幾乎不給予未來新業務任何價值，安全邊際極為充足。

### 護城河分析

綜合金融牌照構成稀缺性壁壘——壽險+產險+銀行+信託+證券+科技全牌照佈局，交叉銷售能力獨一無二。品牌在保險行業認知度第一。但另一方面，地產投資敞口（不動產+地產相關債權）仍是市場最大擔憂，雖已大幅計提減值，但尾部風險未完全消除。

### 估值判斷

極度便宜。P/EV 0.64倍處於歷史最低區間，隱含的假設是平安未來不再創造任何新業務價值——這顯然過於悲觀。即使保守假設NBV增速5%，合理P/EV至少0.8-1.0倍，對應30-50%的上行空間。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★ |
| 估值吸引力 | ★★★★★ |
| **綜合評分** | **★★★★** |

**結論**：綜合金融龍頭估值處於歷史底部，壽險改革拐點已確認。安全邊際充足，值得深度研究。

## 3. 寧波銀行（002142.SZ）

**市值**：約1900億元 | **PE(TTM)**：約6.5倍 | **股息率**：約3.5%

### 核心投資邏輯

城商行中的"異類"——管理機制最市場化，ROE 12.7%在同體量銀行中領先。"大零售+輕資本"戰略執行清晰，消費貸和財富管理雙輪驅動。管理層穩定性和執行力在城商行中獨一無二。

### 護城河分析

管理機制構成差異化競爭力。寧波銀行的核心壁壘不在牌照（城商行牌照不稀缺），而在"類外資行"的治理結構——新加坡華僑銀行為第二大股東，管理層薪酬市場化、考核機制透明、人才梯隊完善。長三角深耕帶來區域稟賦優勢。但另一方面，城商行跨區域擴張存在天然限制，規模天花板低於全國性銀行。

### 估值判斷

便宜。PE 6.5倍，PB約0.9倍。對於一家ROE 12.7%、增速8-10%的銀行而言，估值具有明顯安全邊際。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★★ |
| 估值吸引力 | ★★★★★ |
| **綜合評分** | **★★★★** |

## 4. 興業銀行（601166.SH）

**市值**：約5500億元 | **PE(TTM)**：約5.4倍 | **PB**：約0.51倍 | **股息率**：約5.7%

### 核心投資邏輯

破淨最嚴重的股份行之一，PB僅0.51倍，股息率5.7%。"商行+投行"戰略轉型中，綠色金融和財富管理為發力方向。估值極低提供了足夠的安全邊際，但ROE僅9.2%，盈利能力在股份行中偏弱。適合作為高股息配置標的。

### 護城河分析

同業業務歷史積累的機構客戶資源是差異化優勢，綠色金融品牌在行業中有一定辨識度。但另一方面，興業銀行的護城河不如招行（零售端）清晰，同業業務在監管趨嚴後優勢被削弱，轉型期戰略方向多次調整，尚未形成穩固的新護城河。

### 估值判斷

極度便宜，但可能是"價值陷阱"。PB 0.51倍意味著市場認為其淨資產質量存疑或ROE無法回升。如果ROE能穩定在10%+，當前估值具有明顯修復空間；如果ROE繼續下滑，低估值只是合理反映。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★ |
| 護城河強度 | ★★★ |
| 管理層質量 | ★★★ |

| **綜合評分** | **★★★** |

**結論**：估值極低+高股息，適合防禦性配置。但盈利能力偏弱、護城河不清晰，不優先深度研究。

## 5. 工商銀行（601398.SH）

**市值**：約26000億元 | **PE(TTM)**：約5.5倍 | **股息率**：約4.2%

### 核心投資邏輯

全球最大銀行，總資產超48萬億。確定性極高——不會倒閉、持續分紅、利潤穩定。類債券資產，適合追求穩定現金流回報的配置。但增長有限，淨利潤增速長期在個位數甚至零附近徘徊。

### 護城河分析

規模+牌照+國有背景構成不可逾越的壁壘。工行在對公業務（尤其是大型基建、央企貸款）中的市場地位無可替代。但另一方面，國有大行的經營效率受體制約束，ROE趨勢性下行（從15%+降至10%左右），管理層更多服從政策導向而非股東價值最大化。

### 估值判斷

便宜但增長有限。PE 5.5倍、股息率4.2%，作為"類固收"資產定價合理。不存在明顯低估，也不存在高估。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★ |
| 護城河強度 | ★★★★★ |
| 管理層質量 | ★★★ |
| 估值吸引力 | ★★★★ |
| **綜合評分** | **★★★★** |

**結論**：全球最大銀行，確定性極高但增長有限。適合作為類債券資產配置，不需要深度研究。

## 6. 建設銀行（601939.SH）

### 估值判斷

便宜。PE 5.5倍、股息率4.5%，估值水平與工行相當。作為高股息防禦標的，具有配置價值。

## 7. 東方財富（300059.SZ）

**市值**：約3020億元 | **PE(TTM)**：約25倍 | **利潤增速**：+25.8%

### 核心投資邏輯

網際網路券商龍頭，"流量+牌照"雙重壁壘模式。東方財富網+天天基金網積累了海量散戶流量，再透過券商牌照變現。獲客成本遠低於傳統券商，邊際成本接近零。淨利潤增速25.8%，在券商行業中增長質量最高。

### 護城河分析

流量壁壘+牌照壁壘的組合在A股獨一無二。東方財富網月活數億級別，股吧社群粘性強，使用者遷移成本高。基金代銷規模行業第一（天天基金網）。但另一方面，業績與市場活躍度高度正相關——牛市利潤暴增、熊市利潤銳減，週期性明顯。且網際網路流量獲取模式受監管政策變化影響。

### 估值判斷

合理偏貴。PE 25倍，對應25.8%的增速，PEG約1。考慮到業績週期性（當前處於市場活躍期），實際估值可能高於表面數字。如果市場轉冷，利潤可能回落30-40%，屆時PE將迅速膨脹。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★ |
| 估值吸引力 | ★★★ |
| **綜合評分** | **★★★★** |

**結論**：網際網路券商獨特商業模式，流量壁壘突出。估值合理但需市場活躍度支撐，適合在市場低迷期（即其利潤低谷期）逆向研究。

## 8. 中國人壽（601628.SH）

**市值**：約10400億元 | **PE(TTM)**：約6.7倍 | **P/EV**：約0.72倍

### 核心投資邏輯

中國最大壽險公司，市場份額第一。受益於人口老齡化和保障意識提升的長期趨勢。P/EV 0.72倍估值便宜。但經營效率（NBV margin、代理人產能）系統性弱於平安，管理體制偏國企化，改革力度和速度遜於競爭對手。

### 護城河分析

規模+渠道+品牌構成壁壘。網點覆蓋到縣域級別，代理人隊伍規模最大（雖然人均產能偏低）。國壽品牌在三四線城市認知度極高。但另一方面，壽險行業的護城河更多體現在"管理能力"而非"規模"——規模大但效率低的保險公司，護城河實際不深。

### 估值判斷

便宜。P/EV 0.72倍低於1倍，意味著市場對未來新業務價值給予較低評價。如果改革見效、NBV恢復增長，估值有修復空間。但相比平安（P/EV 0.64倍但經營效率更高），國壽的價效比略遜。

### 評分

## 9. 中國太保（601601.SH）

**市值**：約3370億元 | **PE(TTM)**：約7倍 | **P/EV**：約0.5倍

### 核心投資邏輯

壽險+產險雙輪驅動。產險業務（太保產險COR常年優於行業）提供穩定現金流，壽險業務處於改革轉型期。P/EV僅0.5倍，是主要險企中估值最低的，安全邊際極為充足。管理層近年改革態度積極（新任總裁趙永剛推動"長航行動"轉型）。

### 護城河分析

產險業務綜合成本率（COR）長期優於行業，體現了承保紀律和風控能力。壽險代理人轉型在"三化五最"戰略下穩步推進。但另一方面，太保在品牌力和綜合金融佈局上不如平安，在規模和渠道下沉上不如國壽，定位略尷尬。

### 估值判斷

極度便宜。P/EV 0.5倍是主要上市險企中最低的，意味著市場幾乎將太保的未來新業務價值視為零甚至負值。即使保守假設，當前價格也有顯著安全邊際。價效比在保險板塊中僅次於平安。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★ |
| 護城河強度 | ★★★ |
| 管理層質量 | ★★★★ |

**結論**：P/EV 0.5倍提供極高安全邊際，改革積極，價效比僅次於平安。值得作為保險板塊備選深度研究。
