## 1. 寧德時代（300750）

**市值**：2.1萬億 | **PE(TTM)**：25倍

### 核心投資邏輯

全球動力電池龍頭，市佔率37%+，規模效應和技術迭代速度遠超競爭對手。2025年淨利722億（+42%），儲能業務已成為第二增長曲線，開啟天花板。

### 護城河分析

技術護城河極深：麒麟電池、神行超充、凝聚態電池持續引領行業。規模護城河：產能全球第一，成本優勢明顯。客戶鎖定：繫結特斯拉、寶馬、賓士等全球主流車企，切換成本極高。品牌效應使其在儲能招標中享有溢價。

### 估值判斷

25倍PE對應42%的利潤增速，PEG<1看似合理。但2.1萬億市值意味著未來增速必然放緩，需要判斷長期穩態利潤中樞。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★★ |
| 護城河 | ★★★★★ |
| 管理層 | ★★★★ |
| 估值 | ★★★ |
| 綜合 | ★★★★ |

**結論**：值得深研，關鍵是2萬億市值下增速能否持續，需判斷儲能業務能否再造一個"動力電池級"利潤池。

## 2. 比亞迪（002594）

**市值**：1萬億 | **PE(TTM)**：23倍

### 核心投資邏輯

新能源汽車全球銷量冠軍（427萬輛），垂直整合模式（電池+晶片+整車）構建極強成本控制力。全球化加速，海外收入佔比38.7%，正在複製中國成功路徑。

### 護城河分析

垂直整合是核心壁壘：從礦產到電池到整車，全產業鏈自研自產，成本控制力無人能及。規模效應疊加技術迭代速度，使競爭對手難以追趕。品牌從低端向高階突破（仰望、騰勢），產品矩陣覆蓋5萬-100萬價格帶。但汽車行業天然競爭激烈，護城河寬度不如消費品。

### 估值判斷

23倍PE對應高增速合理，但汽車行業價格戰持續，利潤率能否維持是核心變數。若淨利率穩定在5%+，當前估值具有吸引力。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河 | ★★★★ |
| 管理層 | ★★★★★ |
| 估值 | ★★★ |
| 綜合 | ★★★★ |

**結論**：王傳福執行力一流，關注價格戰對利潤率的長期影響，以及全球化能否順利推進。

## 3. 陽光電源（300274）

**市值**：2850億 | **PE(TTM)**：21倍

### 核心投資邏輯

逆變器全球龍頭+儲能第一大業務（373億/+49%），雙輪驅動。海外收入超60%，享受海外高毛利。光儲一體化解決方案能力構建差異化競爭力。

### 護城河分析

逆變器技術壁壘+品牌認知+全球渠道網路構建競爭優勢。儲能系統整合能力全球領先，大型地面電站專案經驗豐富。但硬體產品終究面臨價格競爭，護城河寬度有限。2026Q1利潤-40%暴露了業績波動性風險。

### 估值判斷

21倍PE對應高增速看似合理，但Q1利潤暴跌說明盈利穩定性不足。需觀察下半年能否恢復，若全年利潤仍增長則當前是買點。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河 | ★★★★ |
| 管理層 | ★★★★ |
| 估值 | ★★★ |
| 綜合 | ★★★★ |

**結論**：好公司遇短期波動，需判斷Q1下滑是一次性還是趨勢性，海外儲能訂單是關鍵跟蹤指標。

## 4. 匯川技術（300124）

**市值**：1800億 | **靜態PE**：42倍 | **2026E PE**：26倍

### 核心投資邏輯

"中國西門子"，工控領域國產替代核心受益者。通用自動化+新能源汽車電驅+人形機器人三重成長邏輯疊加。ROE常年20%+，管理層執行力極強。

### 護城河分析

技術壁壘：伺服系統、變頻器、PLC等核心工控產品自研，效能接近甚至超越外資品牌。客戶粘性強：工控產品一旦嵌入產線，切換成本極高。研發驅動：研發費率10%+，持續拓展品類。人形機器人為遠期期權。但面對西門子、ABB等巨頭，全球化仍是短板。

### 估值判斷

靜態42倍PE偏貴，但若2026年利潤兌現則26倍合理。核心風險是估值已price in了較高預期，容錯空間小。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河 | ★★★★ |
| 管理層 | ★★★★★ |
| 估值 | ★★ |
| 綜合 | ★★★★ |

**結論**：好公司但當前估值偏貴，適合回撥時關注，理想買點在2026E PE 20倍以下。

## 5. 三花智控（002050）

**市值**：1800億 | **PE(TTM)**：37倍

### 核心投資邏輯

熱管理隱形冠軍，全球市佔率超20%。製冷空調+新能源汽車熱管理+資料中心液冷三大賽道同時受益，是少有的跨行業平台型零部件公司。

### 護城河分析

細分領域絕對龍頭：電子膨脹閥全球市佔率超50%，技術壁壘+規模效應構建深厚護城河。客戶覆蓋全球主流車企和空調企業，切換成本高。資料中心液冷為新增長極，受益於AI算力爆發。產品看似簡單但精密製造工藝要求極高，後來者難以複製。

### 估值判斷

37倍PE明顯偏貴，市場已充分定價其成長性。需要等待估值消化或業績超預期才有安全邊際。
