# 07-高階製造與軍工（4/10透過）

> 板塊透過率：4/10 = 40%
> 研究日期：2026-05-17

---

## 1. 三一重工（600031.SH）

**市值**：約1870億元 | **PE(TTM)**：約18-22倍 | **PB**：約2.4倍

### 核心投資邏輯

全球工程機械龍頭，挖掘機全球市佔率第一。海外收入佔比超50%，真正實現了從"中國龍頭"到"全球龍頭"的跨越。數智化轉型（樹根互聯工業網際網路平台）+全球化佈局雙輪驅動。工程機械處於週期底部，國內開工率觸底回升，海外市場持續放量。

### 護城河分析

品牌+渠道+服務網路構成全球化壁壘。三一在全球擁有完善的代理商和服務網路，客戶一旦使用三一裝置，售後服務和配件供應形成鎖定效應。數智化能力（遠端診斷、預測性維護）進一步加深客戶粘性。規模效應使其在成本端擁有優勢。但另一方面，工程機械行業具有明顯週期性，且卡特彼勒、小松等全球巨頭在高階市場仍有技術優勢，三一在利潤率上與國際龍頭仍有差距。

### 估值判斷

週期底部估值合理偏低。PE 18-22倍看似不低，但處於行業週期底部（利潤處於低位），正常化盈利對應的PE可能在12-15倍。PB 2.4倍對比15%+的正常化ROE，具有吸引力。工程機械的週期底部往往是最佳買入時機。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★★ |
| 估值吸引力 | ★★★★ |
| **綜合評分** | **★★★★** |

**結論**：全球工程機械龍頭，週期底部+全球化放量構成雙重催化。管理層梁穩根/向文波團隊執行力極強。值得深度研究。

## 2. 中航光電（002179.SZ）

**市值**：約785億元 | **PE(TTM)**：約28-35倍

### 核心投資邏輯

軍工聯結器絕對龍頭，防務市場市佔率超40%。聯結器是武器裝備的"神經系統"，每一代裝備升級都需要更高密度、更高可靠性的聯結器。技術壁壘+軍品認證壁壘極高，新進入者需要5-8年時間透過全部資質認證。民用領域（通訊、新能源汽車）正在開啟第二增長曲線。

### 護城河分析

雙重壁壘構成極強護城河。技術壁壘：高可靠性聯結器涉及材料、工藝、設計的深厚積累，失敗成本極高（裝備故障可能導致人命損失）。認證壁壘：軍品供應商需透過嚴格的保密資質、質量體系認證和裝機驗證，週期長達數年，一旦匯入很難被替換。但另一方面，軍工行業受國防預算約束，增速上限有限；且2023-2025年處於訂單調整期，業績波動較大。

### 估值判斷

偏貴但可等待。PE 28-35倍處於歷史中高位，對應當前利潤（受訂單節奏影響偏低）。如果2026-2027年軍工訂單回暖、利潤恢復至正常水平，前瞻PE可能降至20-25倍區間。好賽道好公司，但需要等利潤恢復到合理水平再評估價效比。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★★ |
| 管理層質量 | ★★★★ |
| 估值吸引力 | ★★★ |
| **綜合評分** | **★★★★** |

**結論**：軍工聯結器龍頭，技術+認證雙重壁壘極高。當前估值偏貴，建議等利潤恢復後再深度研究。

## 3. 紫光國微（002049.SZ）

**市值**：約590億元 | **PE(2026E)**：約38倍

### 核心投資邏輯

特種晶片（軍用FPGA、安全晶片）龍頭，認證壁壘極高。特種晶片市場進入門檻極高（保密資質+軍品認證+技術積累三重壁壘），一旦產品透過裝機驗證，替換機率極低。毛利率60%+，淨利率40%+，盈利能力在A股晶片公司中頂尖。

### 護城河分析

認證壁壘+技術壁壘構成極強護城河。特種FPGA國內僅2-3家供應商，紫光國微市佔率領先。安全晶片（SIM卡、銀行卡、身份證晶片）市場份額全球前三。但另一方面，業績波動較大——軍品訂單具有"大小年"特徵，2024-2025年處於訂單低谷期，利潤大幅下滑；且特種晶片市場總規模有限（數百億級別），成長天花板存在。

### 估值判斷

偏貴。PE(2026E) 38倍，對應的是利潤低谷後的恢復預期。如果利潤恢復不及預期，估值將面臨壓力。考慮到業績波動性和成長天花板，當前估值缺乏足夠安全邊際。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★★ |
| 護城河強度 | ★★★★★ |
| 管理層質量 | ★★★ |
| 估值吸引力 | ★★ |
| **綜合評分** | **★★★** |

**結論**：特種晶片領域護城河極高，但業績波動大、當前估值偏貴、成長天花板有限。可選擇性研究，非優先標的。

## 4. 綠的諧波（688017.SH）

**市值**：約390億元 | **PE(TTM)**：約429倍

### 核心投資邏輯

國產諧波減速器龍頭，打破了日本哈默納科的長期壟斷。諧波減速器是工業機器人和協作機器人的核心零部件（佔機器人成本10-15%），技術壁壘高、國產替代空間大。人形機器人敘事為公司估值提供了想象空間。

### 護城河分析

技術壁壘存在但並非不可逾越。綠的諧波在精度、壽命等指標上已接近哈默納科水平，並且在成本端有優勢。但另一方面，來歌樂、雙環傳動等國內競爭對手快速追趕，諧波減速器並非"贏家通吃"市場；且傳統工業機器人增速放緩，人形機器人大規模量產至少還需3-5年，短期缺乏利潤支撐。

### 估值判斷

嚴重高估。PE 429倍意味著當前股價完全由人形機器人的遠期預期驅動，而非基本面支撐。即使按最樂觀的2027年預期利潤，PE仍在80-100倍。這是一個"夢想定價"而非"價值定價"的標的。對於價值投資者而言，當前估值不具有任何安全邊際。

### 評分

| 維度 | 評分 |
|------|------|
| 生意質量 | ★★★ |
| 護城河強度 | ★★★★ |
| 管理層質量 | ★★★★ |
| 估值吸引力 | ★ |
| **綜合評分** | **★★** |

**結論**：國產諧波減速器龍頭，技術壁壘真實存在。但429倍PE完全是概念驅動，零安全邊際。暫不建議研究。

| 優先順序 | 公司 | 綜合評分 | 核心理由 |

**板塊觀察**：高階製造與軍工板塊透過率僅40%，主要原因是軍工企業普遍FCF為負（重資產投入）、ROE偏低，不符合"好生意"的篩選標準。透過的4家中，三一重工是唯一在"好公司+好價格"維度均有吸引力的標的，建議優先深研。軍工標的（中航光電、紫光國微）護城河極強但估值偏貴，適合建立觀察清單、等待訂單恢復和估值回落時再介入。
