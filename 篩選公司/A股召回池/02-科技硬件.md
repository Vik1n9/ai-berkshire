## 1. 中際旭創（300308）— 光模組全球龍頭

### 基本資訊

| 指標 | 資料 |
|------|------|
| 市值 | 約9500-9800億元（股價約880元） |
| PE(TTM) | 約64倍 |
| 股息率 | 0.16% |

### 生意質量 ★★★★

商業模式相對清晰：為全球超大規模資料中心提供高速光模組，本質是"賣鏟子"給AI基建。收入隨AI資本開支增長而增長，客戶是谷歌、微軟、Meta等雲巨頭。2025年淨利潤約150億元（估計），2026年機構預期230-260億元，增速極快。

但需注意：光模組本質仍是製造業，技術迭代快（400G→800G→1.6T），每一代產品生命週期約2-3年，需要持續研發投入保持領先。並非真正的"一次投入、持續收割"的簡單生意。毛利率30%+在硬體製造中算優秀，但遠不如軟體或平台型公司。

### 護城河強度 ★★★★

- **技術領先**：800G市場份額超50%，1.6T領先同業6-12個月，"量產一代、研發一代、預研一代"節奏
- **客戶繫結**：深度嵌入谷歌OCS生態，作為光模組核心供應商，客戶轉換成本高
- **規模效應**：全球最大光模組廠商，產能規模和良率控制形成成本優勢
- **但另一方面**：新易盛等競爭對手在快速追趕，光模組行業歷史上出現過龍頭更替（如Finisar被II-VI收購），技術壁壘不是不可逾越

### 管理層質量 ★★★★

實控人王偉修（享國務院特殊津貼），2016年收購蘇州旭創後主動讓位，交由創始人劉聖（中科院碩士+佐治亞理工博士）主導經營。形成"技術專家+職業經理人"的治理結構。管理層在戰略判斷上表現優秀——2016年收購旭創是決定性一步，近年在CPO、矽光等前沿技術上持續佈局。

### 估值吸引力 ★★

當前PE(TTM)約64倍，對應2026年前瞻PE約40-42倍。考慮到利潤增速120%-150%，PEG約0.3-0.5，看起來"不貴"。但問題是：AI資本開支的週期性風險、光模組行業歷史上的週期波動（2018-2019年曾大幅下滑）、以及當前萬億市值是否已充分反映了樂觀預期。市場給予了極高的增長確定性溢價。

### 綜合評分 ★★★★

### 一句話總結

AI光模組賽道的絕對王者，技術領先+客戶繫結構成強護城河，但萬億市值已隱含極高增長預期，適合在行業出現恐慌或增速預期下修時關注。

## 2. 中微公司（688012）— 刻蝕裝置

### 基本資訊

| 指標 | 資料 |
|------|------|
| 市值 | 約2000-2100億元（股價約330元） |
| PE(TTM) | 約93倍（靜態），2026E約64倍 |
| 股息率 | 0.09% |

### 生意質量 ★★★★

半導體裝置是典型的"賣鏟子給淘金者"模式。客戶一旦驗證透過裝置，替換成本極高（需要重新驗證工藝，耗時數月至一年）。刻蝕裝置是晶片製造三大核心環節之一，市場空間大、技術壁壘高。2025年營收約124億元（同比+37%），淨利潤約21億元（同比+31%），增長穩健。

但需注意：半導體裝置行業具有明顯週期性，受下游晶圓廠資本開支驅動。且中微在全球範圍仍面臨泛林（Lam Research）、東京電子等巨頭競爭，國內市場貢獻了絕大部分收入。

### 護城河強度 ★★★★

- **技術壁壘**：刻蝕裝置研發週期5-7年，中微已攻克5nm及以下先進製程，是國內唯一進入國際一線客戶的刻蝕裝置商
- **客戶粘性**：裝置驗證週期長，一旦匯入產線，替換成本極高
- **專利護城河**：公司積累了大量核心專利，且在設計上刻意區別於國際競品，避免侵權同時構築自身壁壘
- **但另一方面**：在全球市場，泛林和東京電子合計佔刻蝕市場70%+份額；中微公司的成長高度依賴國產替代政策推動和地緣政治環境

### 管理層質量 ★★★★★

創始人尹志堯博士（60歲回國創業），此前在應用材料擔任副總裁13年，是行業頂級技術專家。管理團隊杜志遊、倪圖強等均為國際半導體裝置資深人士。公司實行全員持股激勵，核心團隊穩定性極強。尹志堯在2024年提出"未來5-10年覆蓋積體電路關鍵領域50%-60%裝置"的宏大目標，體現了高遠的戰略視野和執行力。

### 估值吸引力 ★★

PE(TTM)約93倍，即使按2026年一致預期淨利潤30.75億元計算，前瞻PE仍有64倍。半導體裝置是公認的好賽道，但市場已給出極高估值。對比全球龍頭泛林（Lam Research）PE約25倍，中微的估值溢價來自"國產替代"邏輯和更高的增速預期。如果國產替代程序放緩或行業週期下行，估值將面臨顯著壓力。

### 綜合評分 ★★★★

### 一句話總結

國產刻蝕裝置的絕對龍頭，管理層是稀缺的頂級技術創業團隊，但當前估值已充分反映國產替代的樂觀預期，需要等待更好的價格視窗。

## 3. 瀾起科技（688008）— 記憶體介面晶片

### 基本資訊

| 指標 | 資料 |
|------|------|
| 市值 | 約2600億元（股價約248元） |
| PE(TTM) | 約118倍（基於EPS 2.11元） |
| 股息率 | 0.28% |

### 生意質量 ★★★★★

記憶體介面晶片是伺服器記憶體條的必備元件，每根DDR5記憶體條都需要配套1顆RCD晶片和多顆資料緩衝晶片。全球僅瀾起和瑞薩兩家供應商，形成寡頭壟斷格局。毛利率高達60%+，淨利率35%，接近晶片設計公司的最佳水平。商業模式簡單清晰：設計晶片→交給代工廠生產→賣給記憶體廠和伺服器廠商。

這是一個真正的"好生意"：需求隨伺服器出貨量和DDR5滲透率增長而增長，市場集中度極高，產品標準化程度高。2026Q1淨利潤8.47億元（同比+61%），全年預期約30億元。

### 護城河強度 ★★★★★

- **雙寡頭壟斷**：全球僅瀾起和瑞薩可提供完整DDR5記憶體介面晶片方案，壁壘極高
- **標準制定者**：參與JEDEC標準制定，是DDR5標準的重要貢獻者
- **認證壁壘**：產品需透過英特爾、AMD平台認證，新進入者面臨數年認證週期
- **技術迭代優勢**：從DDR4到DDR5的成功切換證明了技術延續性
- **但另一方面**：如果未來DDR6標準變化導致技術路線轉換，或者英特爾/AMD自研介面晶片，壟斷格局可能被打破

### 管理層質量 ★★★★

創始人楊崇和博士，1997年首次創業成功（被IDT以8500萬美元收購），2004年創立瀾起。擁有連續創業成功經驗和深厚的半導體行業積累。管理哲學清晰："預判市場趨勢、組建強大團隊、保持資金充裕"。研發人員佔比71%，研發投入約15%（高於同行）。

但需注意：楊崇和持美國國籍（中國永久居留權），在當前地緣政治環境下是否存在潛在風險值得關注。

### 估值吸引力 ★★

當前PE(TTM)約118倍，按2026年預期淨利潤30億元計算，前瞻PE約87倍。即使按更樂觀的2027年預期38億元，前瞻PE仍有68倍。寡頭壟斷的好生意值得溢價，但當前估值意味著即使未來兩年利潤翻倍，PE也還在50倍以上。市場給予了極高的長期成長預期。

### 綜合評分 ★★★★

### 一句話總結

全球記憶體介面晶片雙寡頭之一，生意質量是六家中最佳（高毛利+寡頭壟斷+標準制定），但估值高企，適合長期跟蹤、在市場恐慌時果斷買入。

## 4. 新易盛（300502）— 高速光模組

### 基本資訊

| 指標 | 資料 |
|------|------|
| 市值 | 約5000億元（股價約564元） |
| PE(TTM) | 約40倍（基於2025年利潤約94-98億） |
| 股息率 | 約0.1%（估計） |

### 生意質量 ★★★☆

與中際旭創同一賽道，商業模式類似——為資料中心提供高速光模組。不同之處在於：新易盛聚焦中短距資料中心市場，走LPO（線性驅動光模組）技術路徑，強調成本控制和垂直整合（光晶片製造→封裝→模組全鏈條覆蓋）。

2025年營收約269億元，淨利潤約94-98億元。2026年機構預期淨利潤168-178億元（同比+70-80%），增長極快。毛利率38%，淨利率30%+，在光模組行業中表現優異。

但需注意：相比中際旭創的深度客戶繫結，新易盛的客戶集中度和客戶結構資訊披露較少，且在1.6T高階市場的話語權可能弱於旭創。

### 護城河強度 ★★★

- **垂直整合**：從光晶片到模組的全產業鏈覆蓋，成本優勢明顯
- **技術差異化**：LPO路徑在功耗和成本上有優勢，適合短距應用
- **產能規模**：快速擴產能力強
- **但另一方面**：光模組行業競爭激烈，新易盛在高階市場（1.6T、CPO）的佈局不如中際旭創深入；創始人曾因違規減持被處罰，核心技術人員曾有法律糾紛，對長期信任構成隱憂

### 管理層質量 ★★★

董事長高光榮（中專學歷，1998年創業，2008年聯合創立新易盛）。2023年因違規減持被證監會罰沒3149萬元（隱瞞家族信託持股+超限減持），這在管理層誠信方面是明顯減分項。總經理黃曉雷為技術專家出身（東南大學+電子科大碩士），但早年曾因侵犯商業秘密罪被判刑。

管理層的技術執行力不錯（從10G到800G的多代產品迭代表現出色），但誠信記錄存在瑕疵。段永平的標準是"不和不誠信的人做生意"，這是一個值得警惕的訊號。

### 估值吸引力 ★★★

PE(TTM)約40倍，按2026年預期淨利潤170億元計算，前瞻PE約29倍。在光模組三兄弟中估值最低，且增速不弱，看起來具有一定的估值吸引力。但需考慮管理層風險折價和行業週期性。

### 綜合評分 ★★★

### 一句話總結

光模組賽道的二號種子，垂直整合帶來成本優勢，估值相對合理，但管理層誠信瑕疵是硬傷，不符合"買股票就是買人"的核心原則。

## 5. 天孚通訊（300394）— 光無源器件

### 基本資訊

| 指標 | 資料 |
|------|------|
| 市值 | 約3100億元（股價約400元） |
| PE(TTM) | 約110-115倍（基於近四季利潤約21.7億） |
| 股息率 | 0.42% |

### 生意質量 ★★★★★

天孚通訊的商業模式是光通訊的"賣零件"生意：提供超精密光無源器件和光引擎元件，是光模組廠商的上游供應商。毛利率52%、淨利率38%——這在硬體製造業中極為罕見，接近晶片設計公司的水平。本質上是靠超精密製造工藝構建壁壘的"隱形冠軍"。

2025年營收51.63億元（同比+59%），淨利潤20.17億元（同比+50%）。2026Q1淨利潤4.92億元（同比+46%），增速依然強勁。以2025年對外銷售收入計，公司以11.7%的市場份額位居全球光器件供應商第一。

客戶結構值得關注：2024年僅Fabrinet一家貢獻了61.69%的營收，客戶集中度極高，既是核心競爭力的證明（全球頂級精密製造商選擇天孚），也是風險點。

### 護城河強度 ★★★★

- **超精密製造工藝**：在精密陶瓷、工程塑膠、複合金屬、光學玻璃等領域積累的核心工藝，對手難以短期複製
- **全球第一市場份額**：光器件市場份額11.7%全球第一
- **客戶認證壁壘**：一旦匯入頂級客戶供應鏈，替換成本高
- **但另一方面**：客戶集中度過高（單一客戶佔60%+）是明顯風險，如果Fabrinet轉單或分散供應商，業績將受重大沖擊

### 管理層質量 ★★★★

創始人鄒支農（董事長+總工程師）和歐洋（總經理）夫婦聯合創業，2005年成立。20年專注光通訊精密器件，體現了"專注做好一件事"的經營哲學。公司連續七年榮獲"中國光通訊最具競爭力企業10強"榜首。管理層穩定、專注、低調，符合價值投資者對優秀管理層的期待。

2026年啟動港股上市，顯示管理層在資本運作上有積極規劃。

### 估值吸引力 ★

PE(TTM)高達110-115倍，處於歷史估值94%高分位。即使按2026年預期淨利潤32-33億元計算，前瞻PE仍有約95倍。即使按更樂觀的2027年預期41億元，前瞻PE也在75倍。

生意確實好，但當前價格已完全透支了未來兩年的增長。市場容錯率極低——任何低於預期的季報都可能引發股價大幅調整。

### 綜合評分 ★★★☆

### 一句話總結

光通訊領域的"精密製造隱形冠軍"，生意質量優秀（高毛利+全球第一），但115倍的估值意味著零安全邊際，需耐心等待週期回撥或業績大幅釋放後的合理價位。

## 6. 工業富聯（601138）— AI伺服器（豁免C透過）

### 基本資訊

| 指標 | 資料 |
|------|------|
| 市值 | 約1.05萬億元（股價約64元） |
| PE(TTM) | 約30倍 |
| 股息率 | 1.93% |

### 生意質量 ★★★

工業富聯的商業模式本質是AI伺服器的ODM/OEM代工。全球AI伺服器市場份額超40%，是英偉達GB系列和微軟Azure的核心代工廠。2026Q1營收2511億元（同比+57%），淨利潤106億元（同比+103%），增速驚人。

但需注意關鍵弱點：毛利率僅7%，淨利率僅3.8%（這也是為何去劣篩選需要"豁免C"才透過）。這是典型的"大而不肥"生意——營收萬億級別，但利潤率極薄。每一台伺服器賺的錢很少，靠巨大的出貨量取勝。如果下游需求波動或客戶壓價，利潤彈性有限。

### 護城河強度 ★★★

- **規模壁壘**：年產能超200萬台AI伺服器，全球無出其右
- **客戶深度繫結**：聯合研發切入設計初期，前五大客戶貢獻超60%營收
- **全球化佈局**：多國生產基地分散地緣風險
- **良率與交付能力**：大規模量產的良率控制是核心競爭力
- **但另一方面**：代工模式天然缺乏定價權，核心利潤由品牌方（英偉達、雲廠商）獲取；毛利率7%意味著幾乎沒有議價能力；如果出現競爭對手（如緯創、廣達）產能擴張，利潤率可能進一步承壓

### 管理層質量 ★★★

背靠鴻海/富士康體系，擁有全球製造管理經驗。但作為鴻海集團上市平台，獨立性存疑——公司治理受母公司影響較大。郭台銘時代的強勢管理風格雖然保障了執行力，但也意味著重大決策可能更多服務於集團利益而非A股中小股東利益。

### 估值吸引力 ★★★★

PE(TTM)約30倍，按2026年預期淨利潤515億元計算，前瞻PE約21倍。在六家公司中估值最低，且有近2%的股息率。對於一個營收萬億、利潤增速45%+的公司來說，21倍前瞻PE看起來合理甚至偏低。

但低估值有低估值的道理：3.8%的淨利率意味著代工本質未變，長期ROE天花板有限。

### 綜合評分 ★★★

### 一句話總結

全球AI伺服器代工的絕對規模王者，估值合理且有股息，但7%毛利率的代工生意本質決定了它是"賺辛苦錢"，不符合"好生意"的最高標準。
