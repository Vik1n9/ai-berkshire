# ai-berkshire Roadmap

## P0：近期（1-2個月）

### A股資料來源接入
- 接入 akshare、東方財富等免費資料來源
- 覆蓋 A 股財務資料、行情、龍虎榜
- 現有 Skill 無需改動，資料層擴充套件即可

## P1：中期（3-6個月）

### HTML 報告輸出
- 在 Markdown 基礎上增加 HTML 報告格式
- 支援暗色模式、導航欄、圖表視覺化
- 提升報告傳播性和閱讀體驗

### 多檔深度模式
- `lite`：5分鐘速判，快速給出估值區間和核心結論
- `standard`：當前預設模式，完整多Agent研究
- `deep`：增加更多交叉驗證和歷史類比，機構級深度

### 多股橫向對比
- 支援 2-4 只股票同維度橫向對決
- 同行業公司估值對標
- 輸出對比矩陣和擇優建議

## P2：長期（6個月+）

### 測試覆蓋
- 為核心工具（financial_rigor.py 等）增加單元測試
- 為 Skill 輸出增加回歸測試
- 確保迭代不破壞已有功能

### 組合級分析
- 持倉組合健康度評估
- 行業/地域集中度分析
- 相關性風險檢測
